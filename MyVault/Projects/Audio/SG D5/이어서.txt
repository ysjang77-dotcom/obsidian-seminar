  ★★ PIN 7 = LCD 제어 핀 (74HC245 DIR 추정)                                                                     

  - YM=7 코드가 pin 7을 LOW로 만들어 LCD 데이터 방향 반전 → 흰 화면
  - setup() 맨 앞에 pinMode(7, OUTPUT); digitalWrite(7, HIGH); 추가 → 검정 화면 복원!

  현재 코드 상태

  - pin 7 = OUTPUT HIGH ✓
  - YM = 29 ✓
  - TOUCH_TYPE = 0 ✓
  - procTouch() → 주석 처리 상태 (다음에 재활성화 필요)
