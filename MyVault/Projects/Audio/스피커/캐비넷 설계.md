# 스피커 캐비넷 설계 시뮬레이터

스피커 **캐비넷(인클로저)** 설계/튜닝을 도와주는 시뮬레이터(소프트웨어) 목록과, 어떤 상황에 어떤 툴이 맞는지 정리.

## 1) 자주 쓰는 인클로저 시뮬레이터

### WinISD (무료, Windows)
- 용도: **밀폐(Sealed)** / **베이스리플렉스(Ported)** 설계 입문
- 강점: T/S 파라미터 기반으로 용적, 포트 튜닝, 주파수 응답, 콘 익스커션, 포트 공기속도 등을 빠르게 확인
- 메모: “처음 박스 잡기”에 가장 무난

### Hornresp (무료, Windows)
- 용도: **혼(Front/Back loaded)**, 밴드패스, 로딩이 복잡한 인클로저
- 강점: 다양한 어쿠스틱 로딩/구조를 폭넓게 다룸
- 주의: UI/개념 난이도 높음(입문 후반~)

### VituixCAD (무료, Windows)
- 용도: 시스템 설계(특히 **크로스오버/지향성**) 중심 + 인클로저/배플 효과 포함
- 강점: 드라이버 응답 + 배플 + 필터 + 오프축까지 **통합 워크플로우**
- 메모: “박스만”보단 “스피커 전체(2웨이/3웨이)”를 같이 다루고 싶을 때 유리

### Boxsim (무료, Windows, Visaton 중심)
- 용도: 스피커 시스템/필터 포함 시뮬레이션
- 메모: 데이터/드라이버 환경이 맞으면 편하지만 생태계 의존이 있을 수 있음

### 상용(유료)
- BassBox Pro / LEAP / SoundEasy 등
- 장점: 상용 워크플로우, 데이터 관리, 고급 기능
- 단점: 비용 및 학습 부담

## 2) 특수 케이스

### Transmission Line(TL, 전송선)
- 설계 난이도가 높아 전용/준전용 툴이나 고급 시뮬레이터(AkAbak 계열 등)를 쓰는 경우가 많음
- 목표가 TL이면 “TL에 강한 툴” 기준으로 선정하는 게 안전

### AkAbak / ABEC (스크립트 기반)
- 장점: 자유도 높고 복잡한 모델링 가능
- 단점: 툴이라기보다 시뮬레이션 엔진에 가까워 진입장벽이 큼

## 3) 권장 흐름(현실적인 결과를 위한 체크)

1) 드라이버 스펙 확보
- 최소: T/S 파라미터(예: Fs, Qts, Vas, Re 등)
- 가능하면: 제조사 제공 응답곡선 + 임피던스 + 측정치

2) 시뮬레이터로 1차 박스/튜닝 산정
- 밀폐/포트 타입 결정
- 내부 용적(유효 용적), 포트 직경/길이, 목표 튜닝 주파수 등

3) 물리 구현 고려
- 내부 보강/흡음재/유닛 변위로 유효 용적 변동
- 포트 공기속도(치핑), 포트 길이(실제 구현), 박스 누설/손실

4) 실측으로 검증 후 보정
- REW(Room EQ Wizard) 같은 측정 툴 + 측정 마이크(예: UMIK-1 등)를 쓰면, 시뮬레이션-현실 차이를 빠르게 줄일 수 있음

## 4) 툴 선택 빠른 가이드

- “우퍼 박스(밀폐/포트)부터”: **WinISD**
- “혼/밴드패스/특수 로딩”: **Hornresp**
- “박스 + 배플 + 크로스오버까지 시스템으로”: **VituixCAD**

## 5) 다음에 정해야 할 질문(프로젝트 시작용)

- 목표 타입: 밀폐 / 포트 / 혼 / TL 중 무엇?
- 유닛 크기 및 용도: 예) 5인치 북셀프, 12인치 서브우퍼
- 드라이버 모델명 또는 T/S 파라미터 보유 여부
- 목표 저역(예: -3 dB 지점), 최대 음압/익스커션 한계, 크기 제약

## 6) 탕밴드 4인치 풀레인지용 백로드혼(BLH) 설계 노트

탕밴드(Tang Band) **4인치 풀레인지**로 “백로드혼(Back-Loaded Horn, BLH)” 캐비넷을 설계할 때 필요한 핵심만 압축 정리.

### 6-1. 먼저 현실적인 목표 잡기

- 4인치 풀레인지 + BLH는 “작은 박스로 30~40 Hz” 같은 목표는 현실적으로 어렵고,
	보통은 **70~100 Hz 근처를 ‘바닥’으로** 두고 **중저역의 에너지/질감**을 얻는 쪽으로 설계하는 경우가 많음.
- 더 낮은 저역을 원하면:
	- (1) 박스(혼 길이/마우스)가 커지거나
	- (2) 서브우퍼/헬퍼우퍼를 병행하거나
	- (3) TL/MLTL 계열로 목표를 바꾸는 선택지가 있음.

### 6-2. BLH에 잘 맞는 드라이버 경향(체크리스트)

아래는 “대체로” 유리한 경향(정답이라기보다 선별 가이드).

- $Qts$가 너무 높지 않은 편(대략 **0.2~0.4 구간**에서 설계 사례가 많음)
- $Fs$가 너무 높지 않을수록 저역 목표가 쉬움(4인치에서는 현실적으로 제한)
- $Xmax$가 큰 편이면 저역에서 안전 여유가 늘어남
- 제조사/측정 기반 T/S가 정확할수록 시뮬레이션-실물 오차가 줄어듦

### 6-3. 4인치 BLH의 크기 가늠(대략)

혼의 기본 길이는 목표 컷오프 $f_c$에 대해 대략 1/4파장으로 시작:

$$L \approx \frac{c}{4 f_c}$$

- $c \approx 343\,\text{m/s}$
- 예) $f_c=80\,\text{Hz}$면 $L\approx 1.07\,\text{m}$
- 예) $f_c=100\,\text{Hz}$면 $L\approx 0.86\,\text{m}$

즉, 4인치로 저역을 내리려면 **혼 경로 길이가 0.9~1.3 m 이상**이 되는 경우가 흔하고, 접어서 넣기 때문에 캐비넷이 생각보다 커질 수 있음.

### 6-4. Hornresp로 BLH 설계하는 실무 절차(권장)

Hornresp는 BLH를 “후면 챔버 + 혼(세그먼트) + (선택) 포트/어퍼처” 형태로 모델링해 반복 튜닝하는 데 많이 씀.

1) 드라이버 데이터 준비
- 최소: $Re, Fs, Qes, Qms, Sd, Vas$ (가능하면 $Le, Bl, Mmd$ 등도)
- 탕밴드 4인치는 모델이 많아서 **정확한 모델명(T/S 표)** 확보가 먼저

2) 목표 설정
- 목표 $f_c$ (예: 80~100 Hz)
- 캐비넷 최대 높이/바닥 면적(현실 제약)

3) 혼 형상(세그먼트) “큰 틀”을 먼저 잡기
- 혼 길이 합 $L_{tot}$를 위 식의 1차 근사로 시작
- 플레어(확장)는 처음엔 단순한 형태(예: 콘/지수)로 두고, 응답 리플/임피던스/익스커션을 보며 조정

4) 스로트/압축비(Compression Ratio) 출발점 잡기
- 4인치 풀레인지 BLH는 압축을 과하게 주면 중저역이 나오긴 하지만 **착색/왜곡/피크**가 늘 수 있음
- 출발점(경험적 범위):
	- 스로트 면적 $S_1$을 대략 **0.3~0.7·Sd** 사이에서 시작
	- 압축비는 **1:1 ~ 2.5:1** 정도부터(과한 압축은 피하는 쪽이 안전)

5) 후면 챔버(리어 챔버) 튜닝
- 리어 챔버 용적($V_{rc}$)과 댐핑(흡음)이 **중저역 피크/딥**과 체감 음색에 큰 영향
- 시작점은 크게 두 가지 전략이 있음:
	- (A) 작은 챔버 + 적절 댐핑으로 “혼 결합”을 강조
	- (B) 조금 큰 챔버 + 댐핑으로 “피크를 억제”
- 어떤 값이 맞는지는 드라이버/목표에 따라 달라서 시뮬레이션 반복이 필요

6) 결과를 볼 때 체크 포인트
- 응답: 저역이 내려가도 **리플(굴곡)**이 심하면 실제 청감에서 ‘붕붕/쿵쿵’이 생길 수 있음
- 임피던스: 피크 위치가 목표 튜닝과 일치하는지
- 콘 익스커션: 목표 음압에서 $Xmax$ 초과 여부

### 6-5. 제작/튜닝 팁(실물에서 차이 많이 나는 부분)

- BLH는 시뮬레이터보다 실물에서 **흡음재/누설/재료 강성** 영향이 큼
- 권장 접근:
	- 리어 챔버와 스로트 근처부터 흡음량을 단계적으로 조절
	- 첫 번째 폴드(급격한 방향 전환) 근처는 공진/리플에 민감해서 보강/라운딩/흡음으로 조정
	- 마우스(출구) 방향/바닥과의 거리(바운더리)도 저역 체감에 영향

### 6-6. 참고 키워드(검색/자료 찾기용)

도면/검증된 설계를 찾을 때는 아래 키워드로 검색하면 자료가 많이 나옴(모델별 변형 다수).

- 키워드: `Back Loaded Horn`, `BLH`, `Hornresp BLH tutorial`, `folded horn`, `rear horn fullrange`
- 커뮤니티/자료: `diyaudio back loaded horn`, `Frugal-phile`, `planet10 BLH`
- 공개 설계 이름(변형/파생이 많음): `Frugel-Horn (FH)`, `Austin (A126 계열)`, `BK-12m` 등

### 6-7. 다음 단계(정확 설계를 위해 필요한 것)

아래 2가지만 주시면 “탕밴드 4인치 + BLH”를 Hornresp 기준으로 구체 파라미터(초안)까지 같이 잡을 수 있음.

- 탕밴드 **정확한 모델명**(예: W4-xxxx)
- T/S 파라미터 표(제조사 스펙 또는 실측값)
