## 실험 설계

### 1. PTFE-Al 접착 메커니즘 규명 실험

#### 1.1 표면 분석 프로토콜

| 분석 기법 | 측정 항목 | 목적 |
|-----------|-----------|------|
| **XPS** (X-ray Photoelectron Spectroscopy) | C-F 결합, Al-O, Al-F 결합 | 계면 화학 결합 분석 |
| **SEM/EDS** | 계면 형태, 원소 분포 | 기계적 결합 (interlocking) 확인 |
| **AFM** | 표면 조도, 접착력 맵핑 | 나노스케일 접착 특성 |
| **접촉각 측정** | 표면 에너지 | PTFE 표면 개질 효과 확인 |

#### 1.2 접착력 측정 프로토콜

| 시험 방법 | 표준 | 측정값 | 비고 |
|-----------|------|--------|------|
| **T-Peel Test** | ASTM D1876 | 박리 강도 (N/mm) | 전극-집전체 접착력 |
| **180° Peel Test** | ASTM D903 | 박리 강도 | 계면 파괴 모드 분석 |
| **Stud Pull-off Test** | ASTM D4541 | 인장 접착력 (MPa) | 수직 방향 접착력 |

### 2. 합지 공정 최적화 실험

#### 2.1 실험 변수 (DOE)

| 변수 | 범위 | 단계 |
|------|------|------|
| **온도** | 100°C ~ 200°C | 25°C 간격, 5단계 |
| **압력** | 10 ~ 100 MPa | 10단계 |
| **속도** | 0.5 ~ 5 m/min | 5단계 |
| **PTFE 함량** | 1 ~ 5 wt% | 5단계 |

#### 2.2 실험 매트릭스 (Taguchi L16)

```
┌─────────────────────────────────────────────────────────────┐
│              합지 공정 최적화 DOE                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  독립 변수          →    반응 변수                           │
│  ├─ 온도 (T)              ├─ 박리 강도 (N/mm)               │
│  ├─ 압력 (P)              ├─ 접촉 저항 (mΩ·cm²)            │
│  ├─ 속도 (v)              ├─ 계면 결합 상태 (XPS)           │
│  └─ PTFE 비율             └─ 전극 두께 변화율 (%)           │
│                                                             │
│  최적화 목표:                                                │
│  - 박리 강도 최대화                                          │
│  - 접촉 저항 최소화                                          │
│  - 전극 물성 유지 (다공성, 두께)                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3. 계면 접착 메커니즘 증명 실험

#### 3.1 Al 표면 산화층 역할 규명

| 실험 | 방법 | 목적 |
|------|------|------|
| Al₂O₃ 두께 제어 | 양극산화, 열처리 | 산화층 두께와 접착력 상관관계 |
| XPS depth profile | Ar⁺ 스퍼터링 | 계면 원소 분포 |
| TEM cross-section | FIB 단면 가공 | 나노스케일 계면 구조 |

#### 3.2 PTFE 섬유화 정도와 접착력 관계

| 변수 | 측정 방법 | 예상 관계 |
|------|-----------|-----------|
| Kneading 시간 | SEM 섬유 길이/직경 | 섬유화↑ → 기계적 interlocking↑ |
| 전단 속도 | 토크 측정 | 최적 전단 조건 존재 |
| 온도 | DSC/TGA | PTFE Tg 부근에서 접착 최대 |

## PTFE-Al 접착 메커니즘 (문헌 기반)

### 1. 접착 메커니즘 분류

```
┌─────────────────────────────────────────────────────────────┐
│            PTFE-Al 접착 메커니즘                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 기계적 결합 (Mechanical Interlocking)                   │
│     ├─ PTFE 섬유가 Al 표면 거칠기에 침투                    │
│     ├─ Al 나노다공성 표면과 섬유 얽힘                       │
│     └─ 열압착 시 PTFE 유동으로 표면 충전                    │
│                                                             │
│  2. 화학적 결합 (Chemical Bonding)                          │
│     ├─ Al₂O₃ 표면 -OH기와 상호작용 (약함)                   │
│     ├─ 표면 처리 시 C-O-Al 결합 형성 가능                   │
│     └─ PTFE 자체는 비극성으로 화학 결합 어려움              │
│                                                             │
│  3. 분자간 힘 (van der Waals Forces)                        │
│     ├─ 접촉 면적에 비례                                     │
│     └─ 열압착으로 밀착도 향상 시 증가                       │
│                                                             │
│  ※ 건식 전극에서는 기계적 결합이 지배적                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2. 문헌 보고 접착 향상 방법

| 방법 | 메커니즘 | 접착력 향상 | 출처 |
|------|----------|-------------|------|
| **나노다공성 Al** | 기계적 interlocking | 유의미 향상 | Wiley |
| **이중 바인더 (PTFE+PVDF)** | PVDF 접착력 보완 | 3~5배 | Adv. Funct. Mater. |
| **수소결합 공바인더 (PAA)** | Al₂O₃-OH + PAA-COOH | 유의미 향상 | ScienceDirect |
| **플라즈마 처리** | 표면 에너지 증가 | 6배 이상 | Scientific Reports |
| **열압착 최적화** | PTFE 유동성 증가 | 공정 의존 | Nature Comm. |

### 3. PTFE 특성과 접착 관계

| PTFE 특성 | 값 | 접착에 미치는 영향 |
|-----------|-----|-------------------|
| 표면 에너지 | ~18 mN/m (매우 낮음) | 젖음성 불량, 접착 어려움 |
| 융점 (Tm) | ~327°C | 열압착 온도 상한 |
| 유리전이 (Tg) | ~120°C | 이 온도 이상에서 유동성 증가 |
| 결정화도 | 90% 이상 | 높은 결정성이 섬유화 가능케 함 |

## 혁신성

### 1. 코팅 없는 순수 Al 집전체 사용

| 기존 방법 | 제안 방법 |

## 한계점

### 1. PTFE 본질적 한계

| 한계 | 설명 | 대응 방안 |
|------|------|-----------|
| 낮은 표면 에너지 | 화학적 결합 형성 어려움 | 기계적 결합 극대화 |
| 비극성 | Al₂O₃와 상호작용 약함 | 표면 처리 또는 공바인더 |
| PFAS 규제 이슈 | 환경 규제 강화 추세 | 대체 바인더 병행 연구 |

### 2. 공정 민감도

- 온도/압력 최적 범위가 좁을 수 있음
- 배치 간 재현성 확보 필요
- 대면적 균일 합지 어려움

### 3. 측정 한계

- 계면 결합력 직접 측정의 어려움
- XPS는 표면 ~10nm 정보만 제공
- Peel test 결과 해석의 복잡성

## 저자 언급 (최신 연구 동향)

### 1. 이중 바인더 시스템 (PTFE + PVDF)

> "The synergistic system leverages PTFE's fibrillation-induced cohesion and PVDF's adhesive strength, while the nanostructured aluminum current collector's nanoporous morphology enables robust mechanical interlocking."
> — [Adv. Funct. Mater. 2518597](https://advanced.onlinelibrary.wiley.com/doi/10.1002/adfm.202518597)

### 2. 수소결합 공바인더 (PAA)

> "The introduction of PAA improved adhesion between the dry electrode and the Al current collector through hydrogen bonding between the hydroxyl groups (–OH) in the Al₂O₃ layer and the –COOH groups in PAA."
> — [ScienceDirect S1385894725026154](https://www.sciencedirect.com/science/article/abs/pii/S1385894725026154)

### 3. 나노다공성 Al 집전체

> "In contrast to wet-processed electrodes, dry electrode adhesion relies almost entirely on mechanical bonding, demanding current collector designs tailored specifically for DPEs."
> — [Adv. Funct. Mater. 2518597](https://advanced.onlinelibrary.wiley.com/doi/10.1002/adfm.202518597)

### 4. 열압착 메커니즘

> "During the thermal pressing step, the thermoplastic binder melts and flows around the solid particles, establishing inter-particle adhesion and bonding to the substrate."
> — [Nature Communications](https://www.nature.com/articles/s41467-025-66082-3)

### 3. 접촉 저항과 Peel Test 상관관계

> "A mechanical electrode peel test is often used to compare adhesion and electrical contact resistance."
> — [OSTI.gov](https://www.osti.gov/servlets/purl/1988769)

- Peel 강도 ↑ → 접촉 저항 ↓ 상관관계 존재
- 둘 다 측정하여 교차 검증 권장
