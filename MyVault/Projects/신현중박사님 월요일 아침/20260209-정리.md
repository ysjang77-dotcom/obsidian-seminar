# 20260209 회의 내용 정리

## 1. 온도 조건 관련
- 155도 → 160도로 약 5도 차이 발생
- 170도는 너무 높음, **160도에서 조건 잡는 방향**
- 조건 확정 후 200도로 올려서 표면 안정화 고려 (단, 내부 손실 우려 있음)

## 2. 배합 비율 및 바인더
- 현재 **8:2 비율** → **7:3으로 조정** 검토
- PTF 현재 2% → **3%로 상향** 검토 (바인더 역할로 두께 조절)
- 파우더를 잡아주는 역할: 섬유, CNT, BGCF

## 3. 소재별 제막 비교 (3종)

| 소재 조합 | 제막성 | 압착/두께 |
|-----------|--------|-----------|
| BGCF + CNT | 가장 좋음 | 압착 양호 |
| BGCF only | 보통 | - |
| BGCF + CPRP | 보통 | - |
| Super P | - | 압착 시 두께 감소 어려움 |

## 4. 이번 주 일정
- **수요일까지**: 보고서용 슬럿 스테이지 완료
- **목요일~금요일**: 제막 조건 확정 및 셀 제작
- **설 연휴 후**: 결과 확인
