# ASTM B213 버전별 비교 (B213-97 / 03 / 11 / 17 / 20)

> **표준명**: Standard Test Method(s) for Flow Rate of Metal Powders (Using the Hall Flowmeter Funnel)

> **최초 승인**: 1946년 | **최신 버전**: B213-25 (2025.02)

## 2. 공통 기술사양 (전 버전 동일)

| 항목 | 사양 |
|------|------|
| 깔때기 재질 | 스테인리스 스틸 (내면 연마) |
| 콘 각도 | 60° |
| 오리피스 직경 | 0.1 inch (2.54 mm) |
| 시료 질량 | 50.0 g (±0.1 g) |
| 측정값 | 50 g 전량 통과 시간 (초, s) |
| 저울 정밀도 | 0.1 g 이상 |
| 타이머 정밀도 | 0.1 s 이상 |
| 적용 대상 | Hall Funnel을 자력으로 통과 가능한 금속 분말/혼합물 |
| 교정 기준 | 표준 에머리(Emery) 분말로 40 ± 0.5 s 통과 확인 |

## 3. 버전별 주요 변경사항

### 3.1 B213-97 (1997)

- **기준 버전** (비교 출발점)
- 제목: "Standard Test **Method**" (단수) — 단일 시험법 체계
- **단위 체계**: inch-pound 단위 기본, SI 단위 참고용
- Hall Flowmeter Funnel과 Carney Funnel을 **하나의 표준 내에서** 모두 다룸
- 구조가 비교적 단순 (Scope → Apparatus → Procedure → Calculation → Precision)

### 3.2 B213-03 (2003)

**B213-97 대비 변경점:**

| 항목 | 변경 내용 |
|------|-----------|
| 규모 | **소규모 개정** (editorial revision) |
| 제목 | 변경 없음 — "Standard Test Method" (단수) 유지 |
| 단위 체계 | 변경 없음 — inch-pound 기본 |
| 기술 내용 | 시험 절차의 명확화 및 편집 보정 |
| Carney Funnel | 여전히 본 표준 내에 포함 |

> **핵심**: B213-97과 실질적으로 동일. 편집상 보정 수준.

### 3.3 B213-11 (2011) ⭐ 가장 큰 변화

**B213-03 대비 변경점:**

| 항목 | B213-03 | B213-11 |
|------|---------|---------|
| **제목** | "Standard Test **Method**" | "Standard Test **Methods**" (복수형) |
| **부제** | 없음 | "**Using the Hall Flowmeter Funnel**" 추가 |
| **단위 체계** | inch-pound 기본 | **SI 단위를 기본으로 변경** (깔때기 제작도면만 inch-pound 유지) |
| **Carney Funnel** | 본 표준에 포함 | **별도 표준 ASTM B964로 분리** |
| **시험법 구조** | 단일 시험법 | 복수 시험법 체계 (Methods) |
| **참조 표준** | - | ASTM B964 (Carney Funnel) 신규 참조 |

> **핵심**: 구조적 대개편. Carney Funnel 분리, SI 단위 전환, 복수 시험법 체계 도입.
> Hall Funnel 전용 표준으로 정체성이 명확해짐.

### 3.4 B213-17 (2017)

**B213-13 대비 변경점** (B213-13은 B213-11의 중간 개정판):

| 항목 | 변경 내용 |
|------|-----------|
| 단위 체계 | **inch-pound 단위를 다시 기본으로 복원** (SI 단위는 참고용). 단, 밀도(g/cm³)와 질량(g)은 업계 관행상 유지 |
| Section 9.1.5 | 보정계수(correction factor) 관련 설명 명확화 (B213-13에서 추가) |
| 정밀도/편차 | Precision & Bias 데이터 업데이트 |
| 편집 | 전반적인 용어 정비 및 편집 보정 |

> **핵심**: 단위 체계가 다시 inch-pound 기본으로 회귀. 보정계수 명확화.

### 3.5 B213-20 (2020)

**B213-17 대비 변경점:**

| 항목 | 변경 내용 |
|------|-----------|
| 정밀도/편차 | 최신 실험실간 비교시험(ILS) 데이터 반영 |
| 편집 보정 | 용어, 문구, 참조 표준 번호 업데이트 |
| 기술 내용 | 핵심 시험법 변경 없음 |

> **핵심**: 기술적 변경 최소. 정밀도 데이터 갱신 및 편집 보정.

## 4. 단위 체계 변천사

```
B213-97  → inch-pound 기본 (SI 참고)
B213-03  → inch-pound 기본 (SI 참고)      ← 변경 없음
B213-11  → SI 기본 (inch-pound 참고)       ← SI로 전환 ⭐
B213-13  → inch-pound 기본으로 복귀        ← 다시 복원
B213-17  → inch-pound 기본 (SI 참고)      ← 유지
B213-20  → inch-pound 기본 (SI 참고)      ← 유지
```

> **참고**: B213-11에서 SI 단위로 전환했으나, 업계(분말야금)의 관행과 맞지 않아 B213-13에서 다시 inch-pound로 복귀함.

## 7. 실험 설계 시 고려사항

### 어떤 버전을 사용해야 하는가?

- **신규 실험 설계**: **B213-20** 이상 (현재 최신 B213-25) 사용 권장
- **기존 데이터 비교**: 이전 실험에서 사용한 버전과 동일 버전 유지
- **국제 표준 대응**: ISO 4490과 호환성 확인 필요

### 버전 간 데이터 호환성

- **B213-97 ↔ B213-20**: 핵심 시험법(깔때기 사양, 50g, 측정 방법)은 동일하므로 **데이터 직접 비교 가능**
- 단, 보정계수(correction factor) 적용 방법에 미세한 차이가 있을 수 있음 (B213-13 이후 명확화)
- 정밀도/편차 기준값이 버전마다 다르므로 합/불합격 판정 시 해당 버전 기준 적용 필요

### 파우더 물성 분석 실험과의 연관성

현재 설계 중인 **관직경별 유동 특성 측정 실험**에서:
- Hall Funnel의 **2.54mm 오리피스**를 기본 참조점(baseline)으로 사용
- 다양한 관직경으로 확장 시, B213의 시험 절차를 기본 프로토콜로 채택 가능
- 로드셀 기반 연속 질량 측정은 B213의 이산적(discrete) 50g 통과 시간 측정을 **연속적(continuous)** 방식으로 확장한 것
