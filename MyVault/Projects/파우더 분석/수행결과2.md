## 실험 설계 (보완)

### 1. Orifice Discharge Method (오리피스 배출법)

**표준 기반**: ASTM B213 (Hall Funnel), ASTM B964 (Carney Funnel)

| 항목 | 사양 |
|------|------|
| 오리피스 직경 | 5mm ~ 50mm (5mm 간격, 10단계) |
| 호퍼 형태 | Truncated cone (stem 없음, USP 권장) |
| 호퍼 반각 | 30° ~ 60° (가변) |
| 측정 장비 | 고정밀 로드셀 (0.01g 분해능) |
| 데이터 수집 | 연속 측정 (10Hz 이상) |

**실험 절차**:
1. 파우더를 호퍼에 일정량(예: 50g) 투입
2. 오리피스 개방 후 시간-질량 데이터 연속 기록
3. 질량 유량(g/s) 계산 및 안정화 구간 확인
4. 오리피스 직경별 반복 (최소 3회)
5. 임계 직경(Critical Orifice Diameter) 도출

### 2. Angle of Repose (안식각) 측정

**표준 기반**: ISO 4324, ASTM D6393, ASTM C1444

| 방법 | 표준 | 측정값 |
|------|------|--------|
| 고정 깔대기법 | ISO 4324 | 정적 안식각 |
| 회전 드럼법 | ASTM D6393 | 동적 안식각 |
| 붕괴각 | ASTM D6393 | Angle of Fall |

**유동성 판정 기준** (Carr Index 기반):

| 안식각 | 유동성 |
|--------|--------|
| < 30° | 우수 (Excellent) |
| 30° ~ 45° | 보통 (Fair) |
| > 45° | 불량 (Poor) |

### 3. 밀도 측정

**표준 기반**: ASTM B212, ASTM B527

| 측정 항목 | 표준 | 장비 |
|-----------|------|------|
| 겉보기 밀도 (Apparent Density) | ASTM B212 | Hall Flowmeter Funnel |
| 탭 밀도 (Tap Density) | ASTM B527 | Tap Density Tester |

**압축성 지수 (Compressibility Index)** = (탭밀도 - 겉보기밀도) / 탭밀도 × 100

**Hausner Ratio** = 탭밀도 / 겉보기밀도

| Hausner Ratio | 유동성 |
|---------------|--------|
| < 1.25 | 우수 |
| 1.25 ~ 1.40 | 보통 |
| > 1.40 | 불량 |

### 4. 전단 시험 (마찰 계수 직접 측정)

**표준 기반**: ASTM D6128 (Jenike), ASTM D7891 (FT4)

| 측정 항목 | 기호 | 용도 |
|-----------|------|------|
| 내부 마찰각 | φ | 입자 간 마찰 |
| 벽면 마찰각 | φ_w | 입자-벽면 마찰 |
| 응집력 | c | 호퍼 아칭 예측 |

## 혁신성 (보완)

### 1. 다중 표준 통합 접근

- ASTM B213 (유량) + ASTM D6393 (Carr Index) + ASTM D6128 (전단) 통합
- 단일 파우더에 대해 다각도 물성 프로파일 구축
- 표준 시험 결과와 DEM 파라미터 직접 매핑 가능

### 2. 임계 직경 기반 상대 비교법

- 기존: 전단 시험으로 마찰 계수 직접 측정 (시간 소요, 고가 장비)
- 제안: 임계 직경 비교로 상대적 마찰 특성 신속 스크리닝
- 장점: 간단한 장비, 빠른 측정, 다수 시료 비교 용이

### 3. DEM 역산 분석 (Inverse Analysis)

| 실험 출력 | DEM 입력 역산 |
|-----------|---------------|
| 질량 유량 | 마찰 계수 (가장 효과적) |
| 안식각 | 정적/구름 마찰 계수 |
| 벌크 밀도 | 입자 형상 계수 |

### 4. Hallucination 방지 3-AI 체계

| AI 역할 | 기능 |
|---------|------|
| AI-1 (Executor) | 데이터 수집 및 분석 수행 |
| AI-2 (Validator) | 출처 검증, 표준 규격 확인 |
| AI-3 (Replicator) | 독립 분석으로 교차 검증 |

## 한계점 (보완)

### 1. 표준 간 불일치

- Hall Funnel (ASTM B213) vs Carney Funnel (ASTM B964) 결과 차이
- 전단 셀 종류에 따른 측정값 차이 (Jenike vs FT4 vs Schulze)
- 안식각 측정 방법별 결과 편차 (고정 깔대기 vs 회전 드럼)

### 2. 환경 조건 영향

- 습도: 파우더 응집력에 직접 영향 (ASTM B213 주의사항)
- 온도: 일부 파우더의 유동 특성 변화
- 정전기: 미세 분말의 유동 방해

### 3. 입자 특성 제한

| 조건 | 적용 표준 | 제한사항 |
|------|-----------|----------|
| 자유 유동 분말 | ASTM B213 | 응집성 분말 불가 |
| 응집성 분말 | ASTM B964 | Carney Funnel 필요 |
| 극미세 분말 (<10μm) | - | 특수 처리 필요 |

### 4. DEM 시뮬레이션 한계

- 형상 계수(shape factor) 정확한 측정 어려움
- 접촉 모델 선택에 따른 결과 민감도
- 캘리브레이션 시간 소요 (Cloud Method로 89% 단축 가능)

## 저자 언급 (보완)

### 1. Beverloo 방정식

호퍼 오리피스를 통한 분체 유량:

**W = C · ρ_b · g^0.5 · (D - k·d)^2.5**

| 기호 | 의미 |
|------|------|
| W | 질량 유량 (kg/s) |
| C | 배출 계수 (~0.58) |
| ρ_b | 벌크 밀도 |
| g | 중력 가속도 |
| D | 오리피스 직경 |
| k | 형상 계수 (~1.4) |
| d | 입자 직경 |

### 2. Jenike 설계법 (ASTM D6128 기반)

- 전단 시험으로 Flow Function (FF) 도출
- 호퍼 반각 및 최소 오리피스 직경 설계
- 아칭(arching) 및 랫홀링(ratholing) 방지

### 3. DEM 접촉 모델

| 모델 | 적용 |
|------|------|
| Hertz-Mindlin | 비응집성 구형 입자 |
| Hertz-Mindlin + JKR | 응집성 입자 (습도 영향) |
| Hertz-Mindlin + Rolling Friction | 비구형 입자 |

## 추가 발견 (보완)

### 1. ISO/ASTM 52913-1:2025 (최신)

- 적층 제조(AM)용 분말 유동 특성 측정 가이드라인
- 기존 ASTM B213 등을 AM 분야에 적용하는 방법 제시
- 분말 재사용 시 유동성 변화 평가 포함

### 2. Cloud Method vs Hopper Method

| 방법 | 시뮬레이션 시간 | 정확도 |
|------|-----------------|--------|
| Hopper Method | 기준 (100%) | 높음 |
| Cloud Method | 11% (89% 단축) | 동등 |

### 3. 체적 유량 vs 질량 유량

| 측정 방식 | 장점 | 단점 |
|-----------|------|------|
| 질량 유량 | 측정 용이 | 고밀도 분말 편향 |
| 체적 유량 (ASTM B855) | 금형 충전 조건과 일치 | 측정 복잡 |

### 4. 전단 셀 비교 연구

- Jenike, FT4, Schulze 전단 셀 비교 시 결과 차이 존재
- 동일 시료에 대해 표준화된 프로토콜 필요
- 참고: [Comparison of three rotational shear cell testers](https://www.sciencedirect.com/science/article/abs/pii/S0032591015003010)

## 메모 (보완)

### 실험 시 주의사항

- [ ] 실험 전 파우더 컨디셔닝 (습도 평형, 최소 24시간)
- [ ] Hall Funnel 사용 시 진동 차단 필수
- [ ] 오리피스는 반드시 원형, 버(burr) 없이 가공
- [ ] 파우더 헤드 높이 > 오리피스 직경 × 2.5 유지
- [ ] 최소 3회 반복으로 재현성 확보
- [ ] 온습도 기록 (특히 습도 영향 큰 분말)

### DEM 시뮬레이션 팁

- [ ] Hertz-Mindlin (normal) + Mindlin-Deresiewicz (tangential) 조합
- [ ] 정적 마찰 계수, 구름 마찰 계수가 안식각에 가장 큰 영향
- [ ] 캘리브레이션은 안식각 → 유량 → 밀도 순서 권장
- [ ] Cloud Method로 초기 파라미터 탐색 후 Hopper Method로 정밀화

### 검증 체크리스트

- [ ] 논문 인용 시 원문 확인
- [ ] 표준 규격 최신 버전 확인 (ASTM, ISO 업데이트 주기적)
- [ ] 수치 데이터는 2개 이상 독립 출처로 교차 검증
