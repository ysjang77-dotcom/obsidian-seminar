### 2. 권장 오리피스 직경 사양

#### 2.1 다중 직경 세트 (권장)

| 세트 | 직경 범위 | 간격 | 개수 | 용도 |
|------|-----------|------|------|------|
| **Set A (미세)** | 2.5mm ~ 10mm | 2.5mm | 4개 | 미세 분말, Hall 유사 조건 |
| **Set B (중간)** | 10mm ~ 30mm | 5mm | 5개 | 일반 분말, 임계 직경 탐색 |
| **Set C (대형)** | 30mm ~ 50mm | 10mm | 3개 | 조대 분말, 자유 유동 확인 |

**권장 전체 세트**: 2.5, 5, 7.5, 10, 15, 20, 25, 30, 40, 50mm (총 10개)

#### 2.2 Beverloo 방정식 기반 최소 직경 계산

**Beverloo 방정식**: W = C · ρ_b · g^0.5 · (D - k·d)^2.5

| 파라미터 | 권장값 | 설명 |
|----------|--------|------|
| C | 0.58 | 배출 계수 (범용) |
| k | 1.4~1.5 | 형상 계수 (구형 입자) |
| D_min | > 6d | 최소 오리피스 직경 |
| D_reliable | > 30d | 신뢰성 높은 측정 직경 |

**예시**: 입자 직경 d = 100μm인 경우
- 최소 오리피스: D_min > 6 × 0.1mm = 0.6mm
- 신뢰 직경: D > 30 × 0.1mm = 3mm

### 3. 호퍼 반각(Half-Angle) 권장 사양

#### 3.1 Jenike 설계 차트 기반 권장각

| 벽면 마찰각 (φ_w) | 권장 반각 (θ) | 유동 모드 |
|-------------------|---------------|-----------|
| 10° ~ 15° | **30° ~ 35°** | Mass Flow |
| 15° ~ 20° | **25° ~ 30°** | Mass Flow |
| 20° ~ 25° | **20° ~ 25°** | Mass Flow |
| 25° ~ 30° | **15° ~ 20°** | Mass Flow (경계) |
| > 30° | < 15° | Funnel Flow 위험 |

#### 3.2 실험용 권장 반각

| 용도 | 권장 반각 | 수직 기준 각도 | 비고 |
|------|-----------|----------------|------|
| **표준 시험** | 30° | 60° (콘 각도) | ASTM B213 호환 |
| **Mass Flow 보장** | 20° ~ 25° | 65° ~ 70° | 대부분 분말 Mass Flow |
| **연구용 가변** | 15° ~ 40° | 50° ~ 75° | 유동 모드 전이 연구 |

#### 3.3 Mass Flow vs Funnel Flow 경계


## 마찰 계수 측정 및 DEM 파라미터

### 1. DEM 마찰 계수 전형적 값 (문헌 기반)

#### 1.1 정적 마찰 계수 (μ_s)

| 재료 조합 | 전형적 범위 | 캘리브레이션 목표 |
|-----------|-------------|-------------------|
| 금속 분말 - 금속 분말 | 0.2 ~ 0.5 | 안식각 매칭 |
| 금속 분말 - 스테인리스 벽 | 0.1 ~ 0.4 | 벽면 마찰각 매칭 |
| 세라믹 분말 - 세라믹 분말 | 0.3 ~ 0.6 | 안식각 매칭 |
| 폴리머 분말 - 폴리머 분말 | 0.3 ~ 0.7 | 안식각 매칭 |

#### 1.2 구름 마찰 계수 (μ_r)

| 입자 형상 | 전형적 범위 | 비고 |
|-----------|-------------|------|
| 구형 | 0.01 ~ 0.05 | 형상 효과 최소 |
| 불규칙 | 0.05 ~ 0.2 | 형상 효과 보정 |
| 각진 형상 | 0.1 ~ 0.3 | 높은 구름 저항 |

#### 1.3 캘리브레이션된 값 예시 (문헌)

| 분말 종류 | μ_s (정적) | μ_r (구름) | 출처 |
|-----------|------------|------------|------|
| Ti-6Al-4V (AM용) | 0.3 ~ 0.4 | 0.01 ~ 0.05 | Granular Matter |
| 제약 분말 | 0.2 ~ 0.5 | 0.1 ~ 0.2 | PMC |
| 불꽃 산화제 | 0.288 | - | Springer |

### 2. 마찰 계수 측정 방법

#### 2.1 직접 측정 (전단 시험)

| 표준 | 장비 | 측정값 |
|------|------|--------|
| ASTM D6128 | Jenike Shear Cell | 내부 마찰각 (φ), 벽면 마찰각 (φ_w) |
| ASTM D7891 | FT4 Rheometer | 전단 강도, 유동 함수 |

**마찰 계수 변환**: μ = tan(φ)

| 마찰각 (φ) | 마찰 계수 (μ) |
|------------|---------------|
| 20° | 0.36 |
| 25° | 0.47 |
| 30° | 0.58 |
| 35° | 0.70 |
| 40° | 0.84 |

#### 2.2 간접 측정 (역산 분석)

| 실험 출력 | 역산 입력 | 민감도 |
|-----------|-----------|--------|
| **질량 유량** | 마찰 계수 | **최고** |
| 안식각 | 정적 마찰, 구름 마찰 | 높음 |
| 벌크 밀도 | 형상 계수 | 중간 |

### 3. DEM 캘리브레이션 워크플로우

```
┌─────────────────────────────────────────────────────────────┐
│               DEM 캘리브레이션 워크플로우                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Phase 1: 초기 추정                                          │
│  ├─ 문헌값으로 초기 파라미터 설정                           │
│  ├─ μ_s = 0.3, μ_r = 0.05 (구형 금속 분말 기준)             │
│  └─ 반발 계수 = 0.5                                         │
│                         ↓                                   │
│  Phase 2: 안식각 캘리브레이션                                │
│  ├─ 고정 깔대기 시뮬레이션                                  │
│  ├─ 실험 안식각과 비교                                      │
│  └─ μ_s, μ_r 조정 (Plackett-Burman 설계)                    │
│                         ↓                                   │
│  Phase 3: 유량 캘리브레이션                                  │

│  └─ 파라미터 미세 조정                                      │
│                         ↓                                   │
│  Phase 4: 검증                                              │
│  ├─ 다른 오리피스 직경에서 검증                             │
│  ├─ 벌크 밀도 비교                                          │
│  └─ 오차 < 10% 확인                                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 혁신성

### 1. 임계 직경 기반 상대 비교법 (본 연구 핵심)

| 기존 방법 | 제안 방법 |
|-----------|-----------|
| 전단 시험 (고가 장비, 시간 소요) | 임계 직경 비교 (간단, 빠름) |
| 절대값 측정 | 상대적 순위화 |
| 단일 시료 분석 | 다수 시료 스크리닝 |

**임계 직경 정의**: 유량이 안정화되는 최소 오리피스 직경

### 2. 다중 표준 통합

- ASTM B213 (유량) + ASTM D6393 (Carr Index) + ASTM D6128 (전단) 통합
- 단일 파우더에 대해 다각도 물성 프로파일 구축

### 3. DEM 역산 분석 자동화

| 단계 | 방법 |
|------|------|
| 데이터 수집 | 다중 직경 유량 실험 |
| 모델 학습 | Support Vector Regression (SVR) |
| 역산 | 유량 → 마찰 계수 예측 |

### 4. Hallucination 방지 3-AI 체계

| AI 역할 | 기능 | 검증 방법 |
|---------|------|-----------|
| AI-1 (Executor) | 데이터 수집 및 분석 | 원문 자료 직접 인용 |
| AI-2 (Validator) | 출처 검증 | 2개 이상 독립 출처 교차 확인 |
| AI-3 (Replicator) | 독립 분석 | 동일 결론 도출 여부 확인 |

## 한계점

### 1. 측정 조건 의존성

| 변수 | 영향 | 대응 |
|------|------|------|
| 습도 | 응집력 변화, 유량 감소 | 항온항습 조건 (RH 40-60%) |
| 온도 | 일부 분말 유동 특성 변화 | 온도 기록, 보정 |
| 정전기 | 미세 분말 유동 방해 | 접지, 제전 처리 |

### 2. 입자 특성 제한

| 조건 | 적용 가능 여부 | 대안 |
|------|----------------|------|
| d > 10μm, 자유 유동 | ✅ 적용 가능 | - |
| d < 10μm, 응집성 | ⚠️ 제한적 | Carney Funnel, 공기 보조 |
| 비구형 입자 | ⚠️ 보정 필요 | 형상 계수 측정 |

### 3. DEM 시뮬레이션 한계

- 입자 수 증가 시 계산 시간 급증
- 접촉 모델 선택에 따른 결과 민감도
- 형상 계수 정확한 측정 어려움

## 저자 언급 (구체적 수치)

### 1. Beverloo 방정식 상수

| 상수 | 값 | 출처 |
|------|-----|------|
| C (배출 계수) | 0.55 ~ 0.65 (통상 **0.58**) | Thermopedia |
| k (형상 계수) | ~1.5 (구형 입자) | arXiv |

### 2. Jenike 설계 원칙

- 아칭 방지 최소 출구 직경: **B_min = σ_c · H / ρ_b**
  - σ_c: 비압축 항복 강도
  - H: 호퍼 함수
  - ρ_b: 벌크 밀도

### 3. DEM 캘리브레이션 정확도 목표

| 항목 | 허용 오차 | 출처 |
|------|-----------|------|
| 안식각 | ±1° | PMC |
| 벌크 밀도 | ±10% | Granular Matter |
| 유량 | ±5~10% | Springer |

## 추가 발견

### 1. Cloud Method 효율성

| 방법 | 시뮬레이션 시간 | 정확도 |
|------|-----------------|--------|
| 기존 Hopper Method | 100% (기준) | 높음 |
| Cloud Method | **11% (89% 단축)** | 동등 |

### 2. 오리피스 직경과 유량 관계

**Beverloo 법칙**: W ∝ (D - kd)^2.5

| D/d 비율 | 유량 신뢰도 |
|----------|-------------|
| < 6 | ❌ 측정 불가 (아칭) |
| 6 ~ 30 | ⚠️ 보정 필요 |
| > 30 | ✅ 신뢰 가능 |

### 3. Plackett-Burman 설계 결과

DEM 파라미터 민감도 순위:
1. **정적 마찰 계수 (μ_s)** - 가장 큰 영향
2. **구름 마찰 계수 (μ_r)** - 두 번째
3. 전단 탄성률 (G) - 세 번째

### DEM 시뮬레이션 초기 파라미터 (권장)

```
# 금속 분말 (구형) 초기값
particle_density = 4500  # kg/m³ (Ti-6Al-4V 기준)
particle_diameter = 30e-6  # m (30μm)
static_friction_pp = 0.3  # 입자-입자
static_friction_pw = 0.2  # 입자-벽
rolling_friction = 0.05
restitution = 0.5
youngs_modulus = 1e8  # Pa (계산 효율을 위해 낮춤)
poissons_ratio = 0.3
```

- [ ] Hall Funnel 표준 (2.54mm, 60°) 사양 확인
