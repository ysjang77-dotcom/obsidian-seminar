# AI 도구 모음

## Claude Code 기본 명령어

| 명령어 | 설명 |
|--------|------|
| `/help` | 도움말 보기 |
| `/init` | CLAUDE.md 파일 생성 (프로젝트 컨텍스트) |
| `/config` | 설정 확인/변경 |
| `Shift+Tab` | Plan 모드 전환 |
| `/clear` | 대화 초기화 |
