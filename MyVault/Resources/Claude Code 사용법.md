# Claude Code 사용법

> 작성일: 2026-02-05

## 개요
Claude Code는 Anthropic에서 제공하는 CLI 기반 AI 코딩 어시스턴트이다.
터미널에서 직접 실행하며, 자연어로 대화하면서 코드 작성/수정/분석 등 다양한 작업을 수행할 수 있다.

---

## 1. 설치 및 실행

```bash
# 설치
npm install -g @anthropic-ai/claude-code

# 실행 (작업할 폴더에서)
cd my-project
claude
```

---

## 2. 기본 명령어 (슬래시 명령어)

| 명령어 | 설명 |
|--------|------|
| `/help` | 도움말 및 사용 가능한 명령어 목록 보기 |
| `/clear` | 현재 대화 내용 초기화 |
| `/compact` | 대화 컨텍스트를 요약하여 압축 |
| `/commit` | 변경 사항을 분석하고 Git 커밋 생성 |

---

## 3. 자연어로 할 수 있는 작업들

### 파일 및 폴더 관련
- `"이 폴더의 구조를 설명해줘"` → 프로젝트 디렉토리 구조 분석
- `"Resources 폴더에 파일 만들어줘"` → 파일 생성
- `"이 파일 내용 수정해줘"` → 파일 편집

### 코드 관련
- `"이 코드에서 버그를 찾아줘"` → 코드 분석 및 디버깅
- `"함수를 리팩토링해줘"` → 코드 개선
- `"테스트 코드 작성해줘"` → 테스트 생성

### Git 관련
- `"커밋해줘"` 또는 `/commit` → 커밋 메시지 자동 생성 및 커밋
- `"PR 만들어줘"` → Pull Request 생성

### 검색 및 탐색
- `"이 프로젝트에서 TODO를 찾아줘"` → 코드 내 키워드 검색
- `"이 함수가 어디서 사용되는지 찾아줘"` → 참조 추적

---

## 4. 권한 모드

Claude Code는 작업 실행 전 사용자에게 권한을 요청한다.
- **파일 읽기**: 자동 허용
- **파일 쓰기/생성**: 사용자 승인 필요
- **터미널 명령 실행**: 사용자 승인 필요

승인 시 선택지:
- **Yes** → 이번 한 번만 허용
- **Yes, don't ask again** → 이후 같은 유형 자동 허용

---

## 5. 유용한 팁

- 작업할 프로젝트 폴더에서 `claude`를 실행하면 해당 프로젝트를 자동으로 인식한다.
- `CLAUDE.md` 파일을 프로젝트 루트에 두면 프로젝트별 지침을 설정할 수 있다.
- 한국어로 자연스럽게 질문하면 한국어로 답변해준다.
- 대화가 길어지면 `/compact`로 컨텍스트를 압축하면 성능이 유지된다.
- 한 번에 여러 작업을 요청하는 것보다 하나씩 요청하는 것이 정확도가 높다.
