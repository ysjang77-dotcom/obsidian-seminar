# 관점 균형 분석 스킬 (Perspective Balance)

> **"모든 투자 판단에는 반대 의견이 존재한다."**
> 단일 관점의 분석은 편향을 유발하고, 잘못된 의사결정으로 이어집니다.

---

## 1. 핵심 원칙

### 1.1 절대 규칙 (Zero Tolerance)

| 규칙 | 설명 | 위반 시 |
|:-----|:-----|:--------|
| **Bull/Bear 쌍 필수** | 모든 전망에 낙관/비관 시나리오 모두 제시 | FAIL |
| **확신 금지 표현** | "확실히", "반드시", "틀림없이" 등 사용 금지 | 수정 요청 |
| **단일 출처 판단 금지** | 한 가지 근거로 결론 도출 금지 | FAIL |
| **확률 명시** | 각 시나리오의 발생 확률 추정 포함 | 경고 |

### 1.2 균형 분석 공식

```
모든 투자 판단 = Bull Case + Bear Case + Base Case

Bull Case (낙관): 상황이 예상보다 좋을 때의 시나리오
Bear Case (비관): 상황이 예상보다 나쁠 때의 시나리오
Base Case (기본): 가장 가능성 높은 시나리오
```

## 2. Bull/Bear 분석 템플릿

### 2.1 시장/섹터 전망 템플릿

```markdown
### [섹터/시장명] 전망

#### Bull Case (낙관 시나리오) - 확률: XX%
- **전제 조건**: [이 시나리오가 실현되기 위한 조건]
- **예상 결과**: [수익률, 지수 목표 등]
- **근거**: [출처와 함께 근거 제시]

#### Bear Case (비관 시나리오) - 확률: XX%
- **전제 조건**: [이 시나리오가 실현되기 위한 조건]
- **예상 결과**: [손실률, 하락 목표 등]
- **근거**: [출처와 함께 근거 제시]

#### Base Case (기본 시나리오) - 확률: XX%
- **전제 조건**: [현 상황 유지 또는 점진적 변화]
- **예상 결과**: [가장 가능성 높은 결과]

#### 종합 판단
- **권고**: [투자 권고]
- **신뢰도**: [High/Medium/Low]
- **주요 모니터링 지표**: [향후 확인해야 할 지표]
```

### 2.2 금리/환율 전망 템플릿

```markdown
### [금리/환율] 전망

| 시나리오 | 확률 | 전제 조건 | 예상 수준 | 포트폴리오 영향 |
|:---------|:----:|:----------|:----------|:---------------|
| Bull | XX% | [조건] | [수준] | [영향] |
| Base | XX% | [조건] | [수준] | [영향] |
| Bear | XX% | [조건] | [수준] | [영향] |

**확률 합계**: 100%
```

## 3. 신뢰도 점수 체계

### 3.1 신뢰도 산출 기준

| 신뢰도 | 점수 | 기준 |
|:------:|:----:|:-----|
| **High** | 80-100 | 3개 이상 독립 출처, Bull/Bear 모두 명확한 근거 |
| **Medium** | 50-79 | 2개 출처, 일부 시나리오에 근거 부족 |
| **Low** | 0-49 | 단일 출처, 편향된 분석 |

### 3.2 신뢰도 감점 요인

| 요인 | 감점 |
|:-----|:----:|
| Bull Case만 제시 (Bear 누락) | -30 |
| Bear Case만 제시 (Bull 누락) | -30 |
| 확률 합계 ≠ 100% | -20 |
| 출처 없는 주장 | -20 per item |
| 단정적 표현 사용 | -10 per item |

## 4. 금지 표현 목록 (Blocklist)

### 4.1 과신 표현 (절대 금지)

| 금지 표현 | 대체 표현 |
|:----------|:----------|
| "확실히 ~할 것이다" | "~할 가능성이 높다 (XX%)" |
| "반드시 ~해야 한다" | "~하는 것을 권장한다" |
| "틀림없이 ~다" | "높은 확률로 ~로 예상된다" |
| "무조건 ~" | "조건부로 ~" |
| "절대 ~아니다" | "가능성이 낮다 (XX%)" |

### 4.2 편향 표현 (경고)

| 편향 표현 | 문제 | 대체 |
|:----------|:-----|:-----|
| "모두가 ~라고 한다" | 군중 심리 유도 | "주요 기관들은 ~로 전망" |
| "역사적으로 항상 ~" | 과거 의존 편향 | "과거 데이터에서 XX% 확률로 ~" |
| "이번에는 다르다" | 예외 편향 | "차별점은 ~이나, 리스크도 존재" |

## 5. 체크리스트 (분석 완료 전 필수 확인)

### 5.1 균형 검증 체크리스트

- [ ] 모든 전망에 Bull/Bear/Base Case가 포함되었는가?
- [ ] 각 시나리오의 확률 합계가 100%인가?
- [ ] 과신 표현이 사용되지 않았는가?
- [ ] 각 주장에 출처가 명시되었는가?
- [ ] 반대 의견이 공정하게 다뤄졌는가?

### 5.2 편향 검증 체크리스트

- [ ] 낙관적 시나리오의 근거 수 = 비관적 시나리오의 근거 수 (±1)
- [ ] 특정 자산/섹터에 대한 일방적 선호가 없는가?
- [ ] 최근 성과에 과도한 가중치를 두지 않았는가?
- [ ] 포트폴리오 추천에 다양한 시나리오 대응이 포함되었는가?

## 6. 적용 예시

### 6.1 올바른 예시 (PASS)

```markdown
### 반도체 섹터 전망

#### Bull Case (낙관) - 확률: 35%
- **전제**: AI 투자 가속화, 데이터센터 수요 지속 증가
- **결과**: 섹터 수익률 +25% YoY
- **근거**: [Morgan Stanley, 2026.01] "AI capex remains strong"

#### Bear Case (비관) - 확률: 25%
- **전제**: 재고 조정 지연, 중국 규제 강화
- **결과**: 섹터 수익률 -15% YoY
- **근거**: [Goldman Sachs, 2026.01] "Inventory correction extends"

#### Base Case (기본) - 확률: 40%
- **전제**: 점진적 회복, AI 수요 견조
- **결과**: 섹터 수익률 +10% YoY

**신뢰도**: Medium (75/100) - 출처 2개, 시나리오 균형
```

### 6.2 잘못된 예시 (FAIL)

```markdown
### 반도체 섹터 전망

반도체 섹터는 AI 성장으로 확실히 상승할 것입니다.
투자자들은 반드시 반도체 펀드를 매수해야 합니다.

**문제점**:
- ❌ Bear Case 누락
- ❌ "확실히", "반드시" 과신 표현 사용
- ❌ 출처 없음
- ❌ 확률 미제시
```
