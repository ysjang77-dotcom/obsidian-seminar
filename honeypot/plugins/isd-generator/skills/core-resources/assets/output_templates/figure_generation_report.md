## 파일 위치

### 프롬프트 파일
```
[출력 디렉토리]/prompts/
├── 01_[캡션명].md
├── 02_[캡션명].md
├── ...
└── prompt_index.md
```

### 이미지 파일
```
[출력 디렉토리]/figures/
├── 01_[캡션명].png
├── 02_[캡션명].png
└── ...
```

## 문서 업데이트 가이드

### 이미지 삽입 방법

각 문서에서 `<캡션명>` 위치를 찾아 다음과 같이 이미지를 삽입합니다:

```markdown
기존:
<연구개발 목표 및 비전>

변경:
![연구개발 목표 및 비전](figures/01_연구개발_목표_및_비전.png)
```

## 후속 작업

### 필수 작업
1. 생성된 이미지 품질 검토
2. 실패 항목 재생성 또는 수동 생성
3. 문서에 이미지 삽입

### 선택 작업
- 이미지 크기/해상도 조정
- 색상/폰트 일관성 검토
- 레이아웃 미세 조정
