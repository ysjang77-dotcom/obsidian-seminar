# AI 이미지 생성 프롬프트 작성 템플릿

본 템플릿은 국책과제 연구계획서 이미지 생성을 위한 프롬프트 작성 가이드이다.
**중요**: 모든 내용은 원본 문서에서 추출한 실제 데이터로 채워야 하며, [플레이스홀더] 형식은 사용하지 않는다.

## 렌더링 규칙 (중요)

AI 이미지 생성 시 프롬프트의 텍스트가 그대로 렌더링될 수 있으므로 다음 규칙을 반드시 준수한다:

| 구분 | 설명 | 렌더링 여부 |
|------|------|------------|
| **ASCII 레이아웃** | 구조와 실제 표시될 텍스트만 포함 | ✅ 렌더링됨 |
| **반드시 포함할 텍스트** | 이미지에 표시될 실제 텍스트 목록 | ✅ 렌더링됨 |
| **텍스트 규격** | 폰트 크기/굵기/색상 정의 (테이블 형식) | ❌ 렌더링 안됨 |
| **스타일 정의** | 박스/화살표/색상 스타일 설명 | ❌ 렌더링 안됨 |
| **작성 지침** | 의도/강조점/배치 설명 | ❌ 렌더링 안됨 |

### 금지 패턴

다음 패턴은 이미지에 그대로 렌더링되므로 **절대 사용 금지**:
- `"텍스트" (24pt)` - 폰트 크기가 텍스트로 렌더링됨
- `"텍스트" ← 네이비 배경` - 화살표 주석이 렌더링됨
- `[설명 텍스트]` - 대괄호 내용이 렌더링됨
- `(강조)`, `(크게)` - 괄호 내용이 렌더링됨

## 템플릿 구조 개요

프롬프트는 다음 14개 섹션으로 구성된다:

| 순서 | 섹션명 | 필수 여부 | 설명 |
|------|--------|----------|------|
| 1 | 캡션 및 요약 | 필수 | 캡션 전체 내용 + 포함해야 할 핵심 요소 |
| 2 | 이미지 유형 | 필수 | PPT 스타일, 해상도, 표현 방식 |
| 3 | 색상 팔레트 | 필수 | 4색 HEX 코드 + 사용처 |
| 4 | 전체 레이아웃 | 필수 | 전체 구조 ASCII 다이어그램 |
| 5 | 상단 영역 | 필수 | 타이틀 바 상세 |
| 6 | 좌측 영역 | 필수 | 좌측 영역 ASCII + 내용 |
| 7 | 우측 영역 | 필수 | 우측 영역 ASCII + 내용 |
| 8 | 하단 영역 | 필수 | 하단 영역 ASCII + 내용 |
| 9 | 텍스트 규격 | 필수 | 폰트 크기/굵기/색상 |
| 10 | 그래픽 요소 | 필수 | 박스/화살표/구분선 스타일 |
| 11 | 반드시 포함할 텍스트 | 필수 | 40개 이상 구체적 텍스트 |
| 12 | 참고 데이터 출처 | 필수 | 5개 이상 테이블 |
| 13 | 체크리스트 | 필수 | 기술용어/디자인/구성 체크 |
| 14 | 버전 관리 | 필수 | 생성일/버전/출처 |

## 프롬프트 작성 시작

### 섹션 1: 캡션 및 요약

```markdown
# AI 이미지 생성 프롬프트

> **캡션**: [캡션 원문 그대로 복사]. [캡션이 포함해야 할 핵심 요소들을 나열 - 예: 5년 단계별 추진 일정, 기관별 역할 분담, 기술 개발 프로세스 등]

---
```

**작성 지침**:
- 캡션은 원본 문서에서 그대로 복사
- 핵심 요소는 캡션 주변 문맥에서 추출
- 해당 이미지가 표현해야 할 내용을 구체적으로 나열

---

### 섹션 2: 이미지 유형

```markdown
## 프롬프트

```
[이미지 생성 프롬프트]

■ 이미지 유형
PPT 슬라이드 스타일의 인포그래픽. 정부 국가연구개발 과제 발표자료 수준의 전문적 도식.
가로형(16:9 비율), 고해상도, 벡터 그래픽 스타일.
생성된 그림은 모두 동일한 디자인 테마 및 색상 체계 적용.
[표현 방식 선택: 계층구조도/타임라인/조직도/테이블/차트/흐름도/네트워크 다이어그램]
[핵심 표현 내용 1-2문장: 예) "5년 추진체계와 검출기술 개발 프로세스를 통합한 복합 인포그래픽"]
```

**작성 지침**:
- 표현 방식은 캡션 유형에 따라 선택
- 핵심 표현 내용은 이 이미지가 전달해야 할 핵심 메시지

---

### 섹션 3: 색상 팔레트

```markdown
■ 색상 팔레트 (4색 제한 - 시리즈 통일)
1. 주조색: #1E3A5F (진한 네이비) - 제목, 핵심 박스, 후반기
2. 보조색: #4A90A4 (청록색) - 전반기, 화살표, 프로세스
3. 강조색: #2E7D5A (진한 녹색) - 성과/목표 달성
4. 배경색: #F5F7FA (밝은 회색) - 배경
```

**작성 지침**:
- 기본 팔레트는 그대로 사용 (시리즈 통일)
- 사용자가 커스텀 팔레트를 지정한 경우에만 변경
- HEX 코드는 반드시 포함

### 섹션 4: 전체 레이아웃

```markdown
■ 전체 레이아웃 구성

┌─────────────────────────────────────────────────────────────────┐
│  [상단 타이틀 바 - 네이비 배경]                                    │
│  "이미지 제목 (캡션에서 추출)"                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  [좌측 영역 - 영역명]                   [우측 영역 - 영역명]        │
│  ┌───────────────────────────┐       ┌───────────────────────┐  │
│  │ 좌측 영역 핵심 내용        │       │ 우측 영역 핵심 내용   │  │
│  │ (예: 5단계 기술 플로우)    │       │ (예: 5년 타임라인)    │  │
│  └───────────────────────────┘       └───────────────────────┘  │
│                                                                 │
│  [하단 영역 - 영역명]                                             │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 하단 영역 핵심 내용 (예: 기관별 역할 분담)               │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**작성 지침**:
- 전체 구조를 한눈에 파악할 수 있도록 ASCII로 표현
- 각 영역의 역할을 간략히 명시
- 이미지 유형에 따라 레이아웃 조정

### 섹션 5: 상단 영역 (타이틀)

```markdown
■ 상단 영역 - 타이틀

[네이비 배경, 흰색 텍스트]
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│               검출기술 개발 모식도 및 추진체계                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**작성 지침**:
- 제목은 캡션에서 직접 추출
- 네이비 배경 + 흰색 텍스트 기본

│   • 후보 20개 이상 선정          │

**작성 지침** (렌더링 안됨):
- 각 박스 내용은 원본 문서에서 추출한 실제 데이터
- 기술 용어, 수치, 연차 등을 정확히 표기
- 플레이스홀더 사용 금지 - 모든 내용 구체적으로 채움
- ASCII 박스 내에는 실제 표시될 텍스트만 기재

- 전반기(1-3년차)는 청록색, 후반기(4-5년차)는 네이비
- 5년차는 녹색 강조

### 섹션 9: 텍스트 규격

```markdown
■ 텍스트 규격 (시리즈 통일)
- 대제목: 24pt, Bold, 네이비(#1E3A5F)
- 섹션 제목: 18pt, Bold, 네이비
- STEP 번호: 16pt, SemiBold, 청록색(#4A90A4)
- 년차 표시: 14pt, Bold, 해당 단계 색상
- 소제목: 14pt, SemiBold, 네이비
- 본문: 11pt, Regular, 진회색(#333333)
- 강조텍스트: 12pt, Bold, 녹색(#2E7D5A)
```

### 섹션 10: 그래픽 요소

```markdown
■ 그래픽 요소 (시리즈 통일)
- 모든 박스: 둥근 모서리(radius 8px)
- 그림자: 연한 드롭섀도우
- STEP 박스: 네이비/청록 테두리, 배경색 #F5F7FA
- 년차 박스: 1-3년차(청록 테두리), 4년차(네이비 테두리), 5년차(녹색 테두리, 굵게)
- 화살표: 두께 3px, 청록색, 세로 방향
- 단계 구분: 상단 색상 띠 (전반기-청록, 후반기-네이비)
- 기관 박스: 주관(중심, 크게), 협력(좌우, 작게), 흰색 배경, 연한 테두리
- 체크마크(✓): 녹색
- 구분선: 1px, 연회색(#DDDDDD)
```

### 섹션 11: 반드시 포함할 텍스트

```markdown
■ 반드시 포함할 텍스트 (한글) - 최소 40개

[제목/섹션 관련]
- "검출기술 개발 모식도 및 추진체계"
- "5년 추진체계 일정"
- "검출기술 개발 프로세스"
- "기관별 역할 분담"
