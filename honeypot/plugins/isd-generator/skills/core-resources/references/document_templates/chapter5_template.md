## 섹션별 작성 가이드

### 1. 기존 사업과의 차별성 (해당 시)

목적: 유사한 기존 사업이 있는 경우, 신청과제의 독창성과 차별성을 명확히 제시

작성 원칙:
- 기본 정보 비교표와 기술적 차별성 비교표를 분리하여 작성
- 기술적 차별성은 핵심 기술 요소별로 구체적으로 비교
- 유사중복 검토 결론에서 차별성 요약 및 독창적 가치 강조

기본 정보 비교 항목:
- 사업명
- 연구목적
- 사업수행주체
- 사업기간
- 총 사업비

기술적 차별성 비교 항목 (예시):
- 핵심 기술 방식 (예: 냉동/액화 사이클)
- 운전 조건 (예: 운전온도, 압력)
- 주요 장비/부품
- 성능 지표
- 개발 내용
- 활용 분야

### 2. 과제 유사중복 자체점검 결과

목적: NTIS 검색을 통한 유사과제 조사 결과 명시

필수 기재 항목:
- 유사과제수: N건
- 기수행과제: O 또는 X
- 타인등록과제: O 또는 X

선택 기재 항목 (유사과제가 있는 경우):
- 검색 키워드
- 검색 기간
- 검색 범위

### 3. 참고문헌

목적: Chapter 1~4에서 인용된 모든 출처를 일괄 정리

참고문헌 유형별 형식:

정부 정책 자료:
```
[N] 관계부처 합동, "[정책명]", 연도
[N] 대한민국 정부, "[계획명]", 연도
[N] [부처명], "[보고서명]", 연도
```

시장조사 보고서:
```
[N] [조사기관], "[보고서명]", 연도
[N] MarketsandMarkets, "[보고서명]", 연도
[N] McKinsey & Company, "[보고서명]", 연도
```

학술 논문/기술 보고서:
```
[N] [저자], "[논문명]", [학술지명], 연도
[N] [기관명], "[기술보고서명]", 연도
```

웹사이트/뉴스:
```
[N] [URL]
[N] [저자], "[기사 제목]", [매체명], 날짜
```

기관 내부 자료:
```
[N] [기관명], "[자료명]", 연도
```

### 4. 참여기관 참여의향서

목적: 협력 체계 증빙을 위한 의향서 첨부 위치 표시

표시 형식:
```
<참여기관 참여의향서>
```

또는 기관별 구분 시:
```
<[기관명1] 참여의향서>
<[기관명2] 참여의향서>
```

## 분량 가이드라인

전체 문서 분량: 50-100줄

섹션별 분량 배분:
- 기존 사업과의 차별성 (해당 시): 약 30% (15-30줄)
- 과제 유사중복 자체점검: 약 10% (5-10줄)
- 참고문헌: 약 50% (25-50줄)
- 참여기관 참여의향서: 약 10% (5-10줄)

## 참고문헌 개수 가이드라인

전체 참고문헌: 15-25개

유형별 개수:
- 정부 정책 자료: 3-5개
- 시장조사 보고서: 5-8개
- 학술/기술 자료: 3-5개
- 웹사이트/뉴스: 3-5개
- 기관 내부 자료: 2-3개
