## 1. 비전/목표 유형 예시

### 캡션: `<연구개발 목표 및 비전>`

```
[이미지 생성 프롬프트]

## 이미지 유형
PPT 슬라이드 스타일의 인포그래픽. 정부 국가연구개발 과제 발표자료 수준의 전문적 도식.
가로형(16:9 비율), 고해상도, 벡터 그래픽 스타일.
연구개발 목표와 비전을 계층 구조도로 표현.

## 색상 팔레트 (4색 제한)
1. 주조색: #1E3A5F (진한 네이비) - 최종 목표 박스
2. 보조색: #4A90A4 (청록색) - 세부기술 박스
3. 강조색: #2E7D5A (진한 녹색) - 기대효과
4. 배경색: #F5F7FA (밝은 회색) - 배경

## 전체 레이아웃 구성

┌─────────────────────────────────────────────────────────────────┐
│  [상단 타이틀 바 - 네이비 배경]                                    │
│  "연구개발 목표 및 비전"                                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│              ┌─────────────────────────────┐                    │
│              │     최종 연구 목표          │ ← 네이비 배경        │
│              │  "~기술 개발 및 실증"       │    흰색 텍스트       │
│              └─────────────────────────────┘                    │
│                           │                                     │
│           ┌───────────────┼───────────────┐                     │
│           ↓               ↓               ↓                     │
│     ┌──────────┐   ┌──────────┐   ┌──────────┐                 │
│     │ 세부기술1 │   │ 세부기술2 │   │ 세부기술3 │ ← 청록색 테두리  │
│     │          │   │          │   │          │                 │
│     │ 핵심내용 │   │ 핵심내용 │   │ 핵심내용 │                 │
│     └──────────┘   └──────────┘   └──────────┘                 │
│           │               │               │                     │
│           └───────────────┼───────────────┘                     │
│                           ↓                                     │
│              ┌─────────────────────────────┐                    │
│              │       기대효과/비전         │ ← 녹색 배경          │
│              └─────────────────────────────┘                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

## 반드시 포함할 텍스트 (한글)
- "연구개발 목표 및 비전"
- "[최종 연구 목표 문구]"
- "세부기술 1: [기술명]"
- "세부기술 2: [기술명]"
- "세부기술 3: [기술명]"
- "[기대효과/비전 문구]"

## 그래픽 요소
- 모든 박스: 둥근 모서리(radius 8px)
- 그림자: 연한 드롭섀도우
- 화살표: 두께 3px, 청록색, 수직 방향
- 최종 목표 박스: 크게 (다른 박스의 1.5배)

## 텍스트 규격
- 대제목: 24pt, Bold, 네이비(#1E3A5F)
- 최종목표: 18pt, Bold, 흰색
- 세부기술 제목: 14pt, Bold, 네이비
- 세부기술 내용: 11pt, Regular, 진회색(#333333)
- 기대효과: 14pt, Bold, 흰색
```

## 2. 로드맵 유형 예시

### 캡션: `<기술개발 로드맵>`

```
[이미지 생성 프롬프트]

PPT 슬라이드 스타일의 인포그래픽. 정부 국가연구개발 과제 발표자료 수준의 전문적 도식.
가로형(16:9 비율), 고해상도, 벡터 그래픽 스타일.
5년 기술개발 로드맵을 타임라인 형식으로 표현.

3. 강조색: #2E7D5A (진한 녹색) - 최종 성과
4. 배경색: #F5F7FA (밝은 회색) - 배경

│  │ 기반   │ 기술   │ 실증   │  │ 고도화 │ 통합   │             │
│  │ 구축   │ 개발   │ 검증   │  │        │ 실증   │ ← 녹색 강조  │
│  ├────────┼────────┼────────┤  ├────────┼────────┤             │
│  │        │        │        │  │        │        │             │

│  │        │        │        │  │        │        │             │
│  └────────┴────────┴────────┘  └────────┴────────┘             │

│                                                                 │
└─────────────────────────────────────────────────────────────────┘

## 반드시 포함할 텍스트 (한글)
- "기술개발 로드맵"
- "전반기 (1~3년차)", "후반기 (4~5년차)"
- "1년차", "2년차", "3년차", "4년차", "5년차"
- "1단계", "2단계"
- 각 연차별 핵심 활동
- 성과물 목록

## 그래픽 요소
- 연차 박스: 둥근 모서리(radius 8px)
- 전반기 박스: 청록색 테두리
- 후반기 박스: 네이비 테두리
- 5년차 박스: 녹색 테두리 (굵게)
- 단계 구분: 상단 색상 띠

## 텍스트 규격
- 대제목: 24pt, Bold, 네이비(#1E3A5F)
- 단계명: 14pt, Bold, 해당 색상
- 연차 번호: 16pt, SemiBold
- 핵심 활동: 12pt, Bold
- 세부 내용: 11pt, Regular

## 3. 추진체계 유형 예시

```

## 색상 팔레트 (4색 제한)
1. 주조색: #1E3A5F (진한 네이비) - 주관기관
2. 보조색: #4A90A4 (청록색) - 협력/위탁기관
3. 강조색: #2E7D5A (진한 녹색) - 역할 강조
4. 배경색: #F5F7FA (밝은 회색) - 배경

## 전체 레이아웃 구성

┌─────────────────────────────────────────────────────────────────┐
│  [상단 타이틀 바 - 네이비 배경]                                    │

├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                  ┌─────────────────────────┐                    │

│          ┌─────────────────┼─────────────────┐                  │

│    ┌──────────┐     ┌──────────┐     ┌──────────┐              │
│    │ 협력기관1 │     │ 협력기관2 │     │ 위탁기관  │ ← 청록색     │

- 각 기관명
- 기관별 역할 목록

## 그래픽 요소
- 주관기관 박스: 네이비 배경, 크게 (1.5배)
- 협력/위탁기관 박스: 청록색 테두리
- 연결선: 실선 2px (주관-협력), 점선 1px (협력 간)
- 둥근 모서리(radius 8px)

## 텍스트 규격
- 대제목: 24pt, Bold, 네이비
- 기관 유형: 14pt, Bold
- 기관명: 16pt, SemiBold
- 역할: 11pt, Regular
