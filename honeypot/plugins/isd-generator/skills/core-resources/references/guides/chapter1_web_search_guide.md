# Web Search Guide

## 개요

Chapter 1 작성을 위한 포괄적 웹 조사 가이드. 정책, 시장, 기술, VoC 4개 영역에 대한 체계적인 검색 전략과 검색어 템플릿을 제공한다.

---

## 검색 영역별 가이드

### 1. 정부 정책 조사

#### 목적
연구개발 과제와 정부 정책의 부합성을 입증하기 위한 근거 자료 수집

#### 검색 대상 정책 (우선순위순)

1. **123대 국정과제** (현 정부)
   - 검색어: `"123대 국정과제" [기술명]`
   - 검색어: `"국정과제" [분야] 2025`
   - 출처: 국정기획자문위원회, 대통령실

2. **12대 국가전략기술**
   - 검색어: `"12대 국가전략기술" [기술명]`
   - 검색어: `"국가전략기술" [분야] 로드맵`
   - 출처: 과학기술정보통신부

3. **과학기술 기본계획**
   - 검색어: `"제5차 과학기술기본계획" [분야]`
   - 검색어: `"과학기술 기본계획" [기술명]`
   - 출처: 과학기술정보통신부

4. **국가연구개발 투자방향**
   - 검색어: `"2026년도 국가연구개발 투자방향" [분야]`
   - 검색어: `"R&D 투자방향" [기술명] 2026`
   - 출처: 과학기술정보통신부

5. **분야별 이니셔티브/전략**
   - 검색어: `"[분야] 이니셔티브" 2024 2025`
   - 검색어: `"[분야] 전략" 과기부 OR 산업부`
   - 예시: "AI 이니셔티브", "양자 이니셔티브", "수소 전략"

#### 검색어 템플릿

```
[기본 검색어]
"[기술명] 정부 정책 2025"
"[분야] 국정과제"
"[분야] 국가전략기술"

[심화 검색어]
"[기술명] site:msit.go.kr"  (과기부)
"[분야] site:motie.go.kr"   (산업부)
"[기술명] site:ntis.go.kr"  (NTIS)

[특정 정책 검색]
"제1차 국가전략기술 육성 기본계획" [분야]
"신성장 4.0 전략" [기술명]
"3대 게임체인저기술" [분야]
```

#### 수집 정보
- 정책명, 발표기관, 발표연도
- 관련 추진전략/과제 내용
- 본 과제와의 부합점
- 부합성 강도 평가 (매우 강함/강함/보통)

### 2. 시장 동향 조사

#### 목적
기술/산업 분야의 시장 규모, 성장 전망, 경쟁 현황 파악

#### 검색 대상

1. **시장 규모 및 전망**
   - 검색어: `"[분야] 시장 규모" 2030 전망`
   - 검색어: `"[기술] market size forecast"`
   - 검색어: `"[분야] CAGR growth"`

2. **산업 동향 보고서**
   - 검색어: `"[분야] 산업 동향" 2024 2025`
   - 검색어: `"[기술] industry report"`

3. **글로벌 시장조사기관 보고서**
   - 검색어: `"MarketsandMarkets [기술]"`
   - 검색어: `"Frost Sullivan [분야]"`
   - 검색어: `"Statista [기술] market"`

4. **국내 조사기관 보고서**
   - 검색어: `"[분야] site:kiet.re.kr"` (산업연구원)
   - 검색어: `"[분야] site:kistep.re.kr"` (KISTEP)

#### 검색어 템플릿

```
[국내 시장]
"[분야] 시장 규모 전망 2030"
"[기술] 국내 시장 성장률"
"[분야] 시장 동향 보고서 2024"

[글로벌 시장]
"[기술] global market size forecast"
"[분야] market growth rate 2030"
"[기술] industry analysis report"

[조사기관별]
"[분야] MarketsandMarkets"
"[기술] Frost Sullivan report"
"[분야] 산업연구원 KIET"
"[기술] KISTEP 보고서"
```

#### 수집 정보
- 시장 규모 (연도별, 단위 포함)
- 성장률 (CAGR)
- 주요 플레이어/선도기업
- 지역별 시장 점유율
- 출처 정보 (보고서명, 기관, 연도, URL)

### 3. 기술 동향 조사

#### 목적
국내외 기술 수준 비교, 기술 격차 분석, 선도 기관/기업 현황 파악

#### 검색 대상

1. **기술수준평가**
   - 검색어: `"[기술] 기술수준평가" site:kistep.re.kr`
   - 검색어: `"[분야] 기술 수준 비교" 국내 국외`

2. **TRL (기술성숙도)**
   - 검색어: `"[기술] TRL" technology readiness level`
   - 검색어: `"[분야] 기술성숙도" 국내`

3. **선도 기업/기관**
   - 검색어: `"[기술] 선도기업" 현황`
   - 검색어: `"[분야] leading companies"`
   - 검색어: `"[기술] R&D 투자" 기업`

4. **기술 로드맵**
   - 검색어: `"[분야] 기술 로드맵" 2030`
   - 검색어: `"[기술] technology roadmap"`

#### 검색어 템플릿

```
[기술 수준]
"[기술] 국내 기술 수준" 선진국 대비
"[분야] 기술 격차" 분석
"[기술] technology gap Korea"

[선도 현황]
"[기술] 선도기업 순위"
"[분야] 글로벌 리더" 기업
"[기술] 특허 출원 현황"

[기술 발전]
"[분야] 기술 발전 전망"
"[기술] emerging technology trend"
"[분야] breakthrough technology 2025"
```

#### 수집 정보
- 국내/국외 기술 수준 비교 (%, 연도 격차)
- 선도 기업/기관 목록
- 핵심 기술 요소별 수준
- 기술 발전 로드맵
- 출처 정보

### 4. 문제점 및 VoC 조사

#### 목적
기존 기술의 한계점 파악, 산업계 수요/요구사항 수집

#### 검색 대상

1. **기존 기술 한계**
   - 검색어: `"[기술] 한계점" 문제점`
   - 검색어: `"[분야] 기술 과제" 해결`

2. **산업계 요구사항**
   - 검색어: `"[분야] 수요기업" 요구사항`
   - 검색어: `"[기술] 산업계" 니즈`
   - 검색어: `"[분야] VoC" 기업`

3. **인력/인프라 현황**
   - 검색어: `"[분야] 인력 부족" 현황`
   - 검색어: `"[기술] 전문인력" 양성`

4. **기업 인터뷰/설문**
   - 검색어: `"[분야] 기업 인터뷰" 요구`
   - 검색어: `"[기술] 설문조사" 결과`

#### 검색어 템플릿

```
[문제점]
"[기술] 기존 한계" 극복
"[분야] 문제점" 분석
"[기술] 기술적 과제"

[VoC]
"[분야] 수요기업 인터뷰"
"[기술] 산업계 의견"
"[분야] 기업 요구사항 조사"

[인력/인프라]
"[분야] 전문인력 부족"
"[기술] 인프라 현황"
"[분야] 교육 양성"
```

#### 수집 정보
- 기존 기술 문제점 (3-5개)
- VoC (기업명, 요구사항, 출처)
- 인력/인프라 현황 데이터
- As-is vs To-be 비교 포인트

## 검색 결과 정리 템플릿

### 정책 조사 결과

```markdown
## 정책 조사 결과

### 1. [정책명]
- 발표기관: [기관명]
- 발표연도: [연도]
- 관련 내용: [정책 내용 요약]
- 본 과제 부합점: [부합 내용]
- 부합성 강도: [매우 강함/강함/보통]
- 출처 URL: [URL]

### 2. [정책명]
...
```

### 시장 조사 결과

```markdown
## 시장 조사 결과

### 1. [시장 분야]
- 시장 규모: [금액] ([연도])
- 예상 성장률: [CAGR %]
- 전망: [연도]년 [금액] 예상
- 주요 기업: [기업 목록]
- 출처: [보고서명], [기관명] ([연도])
- URL: [URL]
```

### 기술 조사 결과

```markdown
## 기술 조사 결과

### 1. [기술 분야]
- 국내 기술 수준: [선진국 대비 %]
- 기술 격차: [연도]년
- 선도 기업/기관: [목록]
- 핵심 기술 요소: [목록]
- 출처: [보고서명], [기관명] ([연도])
```

| [기업2] | "[요구사항]" | [출처] |

## 검색 우선순위

1. **정부 공식 문서** (최우선)
   - .go.kr 도메인
   - NTIS, 국가정책정보포털

2. **공인 연구기관 보고서**
   - KISTEP, KIET, KEIT
   - 출연연구기관 발표자료

3. **글로벌 조사기관**
   - MarketsandMarkets, Frost & Sullivan
   - Gartner, IDC

4. **언론 보도** (보조 자료)
   - 전문 매체 기사
   - 인터뷰/논평 인용

---

## 검색 시 주의사항

1. 최신 데이터 우선 (2023-2025년)
2. 공신력 있는 출처 선택
3. 수치 데이터는 반드시 출처 명시
4. VoC는 실제 기업명과 구체적 내용 포함
5. 검색 결과는 chapter1_references.md에 정리
