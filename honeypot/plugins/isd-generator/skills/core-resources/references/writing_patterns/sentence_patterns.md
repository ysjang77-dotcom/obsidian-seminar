# 문장 구조 패턴

## 개요
ISD 연구계획서에서 자주 사용되는 문장 구조 패턴을 제시합니다. 이 패턴들은 개발 목표, 필요성, 성과, 협력 관계를 명확하게 표현하기 위한 표준 문장 형식입니다.

## 패턴 1: 개발 목표 진술문

### 구조
`[기술명]` 기반 `[적용대상]`의 `[목표상태]` 개발

### 설명
기술 개발의 최종 목표를 명확하게 진술하는 패턴입니다. 어떤 기술을 기반으로, 무엇에 적용하며, 어떤 상태를 목표로 하는지를 순차적으로 표현합니다.

### 사용 예시
- `[기술명]` 기반 `[적용분야]`의 `[성능지표]` 달성 개발
- `[핵심기술1]`을 활용한 `[대상시스템]`의 `[개선목표]` 구현
- `[기술명]` 기반 `[산업분야]` 적용을 위한 `[최종산출물]` 개발

## 패턴 2: 필요성 진술문

### 구조
`[분야]`에서 `[문제점]`을 해결하기 위해 `[해결방안]`이 필요하다

### 설명
현재의 문제점을 명시하고, 그것을 해결하기 위한 기술적 접근의 필요성을 강조하는 패턴입니다.

### 사용 예시
- `[산업분야]`에서 `[현재의 한계]`를 극복하기 위해 `[기술명]` 기반의 `[솔루션]`이 필요하다
- `[적용분야]`의 `[주요과제]`를 해결하기 위해 `[기술적접근]`이 시급하다
- `[시장]`에서 `[경쟁력약화요인]`을 보완하기 위해 `[차별화기술]` 개발이 필수적이다

## 패턴 3: 성과 진술문

### 구조
`[지표명]` `[수치]`% 향상 달성 / `[성과물]` `[수량]`개 개발

### 설명
연구 결과의 정량적 성과를 명확하게 표현하는 패턴입니다. 구체적인 수치와 지표를 포함하여 성과의 규모를 전달합니다.

### 사용 예시
- `[성능지표]` `[목표수치]`% 향상 달성
- `[산출물명]` `[개수]`개 개발 완료
- `[기술명]` 기반 `[적용사례]` `[건수]`건 확보
- `[효율성지표]` `[개선율]`% 개선

## 패턴 4: 협력 진술문

### 구조
`[주관기관]`은 `[협력기관]`과 협력하여 `[협력내용]`을 수행한다

### 설명
연구 수행 기관 간의 역할 분담과 협력 관계를 명확하게 표현하는 패턴입니다.

### 사용 예시
- `[주관기관]`은 `[협력기관1]`과 `[협력기관2]`와 협력하여 `[기술개발]`을 주도한다
- `[대기업A]`는 `[중견기업B]`와 `[연구기관C]`와 함께 `[사업화]`를 추진한다
- `[주관기관]`은 `[위탁기관]`에 `[특정업무]`를 위탁하고 `[자문단]`의 기술 자문을 받는다
