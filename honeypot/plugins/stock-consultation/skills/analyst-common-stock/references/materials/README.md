# Materials Folder

이 폴더는 사용자가 수집한 투자 관련 자료를 보관하기 위한 예시 폴더입니다.

## 용도

- 투자 관련 메모를 마크다운 형식으로 작성
- 섹터 분석, 종목 조사, 뉴스 정리 등을 저장
- `materials-organizer` 에이전트가 이 폴더의 `.md` 파일들을 읽고 정리

## 사용 방법

1. 이 폴더에 `.md` 파일을 작성
2. `stock-consultant` 호출 시 `materials_path` 파라미터로 이 폴더 경로 전달
3. `materials-organizer`가 자동으로 파일을 읽고 요약/분류/키포인트 추출

## 예시 파일

```markdown
# 반도체 섹터 전망.md

2026년 반도체 업황 회복 전망.
- 메모리 가격 상승 추세
- AI 칩 수요 급증
```

## 주의사항

- `.md` 파일만 지원 (v1.0)
- 중첩 폴더는 스캔하지 않음 (1단계만)
- 파일은 Read-only로만 사용 (수정/삭제 금지)
