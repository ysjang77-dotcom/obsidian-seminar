## 1. 설계 전문 AI Agent 및 다중 AI Agent 협업 플랫폼 개발

### 1.1 기계요소 설계 엔지니어링 데이터 확보
- ISO, KS, AGMA, DIN 등 국제/국내 설계 표준 문서 확보 및 DB 구축
- 다양한 문헌 조사 및 프로젝트 설계 이력 등 엔지니어링 데이터 구축
- 기계요소용 재료 물성치, 열처리 조건별 데이터 등 기계소재 데이터 수집 인프라 구축

### 1.2 기계요소 설계 S/W 기술 개발
- 기존 설계 SW 분석: KISSsoft, MASTA 등 기능 분석, 자동화 대상 도출
- 평기어 설계 모듈: ISO 6336 기반 강도계산, 형상설계 프로토타입
- 볼베어링 설계 모듈: ISO 281 기반 수명계산 프로토타입
- MCP 서버 설계: AI Agent 연동 프로토콜 및 인터페이스 정의

### 1.3 (플랫폼) AI Agent 플랫폼 아키텍처 설계 및 인프라 구축
- 플랫폼 아키텍처: 로컬 중심 서비스 아키텍처 설계
- 개발 환경 구축: Agent 개발 프레임워크 선정 등
- 노코드 빌더 기초 설계: Agent 구성 인터페이스 요구사항 정의, UI/UX 설계
- GPU 서버 구축: LLM 학습/추론용 GPU 서버 구성

## 2. 설계/해석 SW 기술 패키징 및 도메인 특화 LLM 구축

### 2.1 기계요소 설계 엔지니어링 지식그래프/온톨로지 프로토타입 구축
- 온톨로지 스키마 설계: 기계요소 도메인 스키마, 파라미터 관계 모델링
- 지식그래프 구축: 기어류, 베어링 지식그래프, 요소 간 연결관계
- 설계 룰 체계화: 설계 의사결정 룰 추출, 제약조건 라이브러리 구축

### 2.2 기계요소 설계 S/W 라이브러리 구축
- 유성기어 설계 모듈: 강도계산, 형상설계, 최적화 기능
- 베벨기어 설계 모듈: 스트레이트/스파이럴 베벨, 제원 설계, 강도 평가
- 웜기어 설계 모듈: 싱글인벨로핑 웜, 제원 설계, 강도 평가
- 롤러베어링 모듈: 테이퍼/원통/니들 롤러베어링 수명/효율 평가
- 축 설계 모듈: 강도/강성 해석, 피로수명 예측

### 2.3 AI-Ready Data를 활용한 도메인 특화 LLM 구축
- 학습 데이터 전처리: 설계 문서 정제, Q&A/Instruction 데이터셋 생성
- 베이스 모델 선정: Llama, Qwen 등 오픈소스 LLM 벤치마킹
- Fine-tuning: 설계 도메인 SFT, LoRA 기반 효율적 학습
- RAG 시스템: 벡터 DB, 설계문서 임베딩, 검색증강생성 기능 개발

### 2.4 (플랫폼) 노코드 AI Agent 빌더 1.0 (파일럿) 개발
- 노코드 빌더 UI 개발: 드래그앤드롭 Agent 구성, 비주얼 워크플로우 에디터
- Agent 템플릿 시스템: 기본 Agent 템플릿, 프롬프트 도구 및 커스터마이징 가이드
- 로컬 데스크톱 앱: Electron/Tauri 기반 로컬 구동 앱 개발
- 대화형 UI: 채팅 기반 설계 지시 인터페이스
