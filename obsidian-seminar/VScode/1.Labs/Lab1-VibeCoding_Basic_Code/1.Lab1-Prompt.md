# 프롬프트 (Gemini 2.5 Pro 기준)

### 선형 회귀 분석 코드 생성을 위한 프롬프트

[목표]
Python을 사용하여 주어진 조건에 따라 데이터를 생성하고, 단순 선형 회귀 분석을 수행하는 예제 코드를 생성해 주세요.

**[분석 과정 및 코드 요구사항]**

1. **데이터 생성:**
    
    - 독립 변수 **X**는 **5, 6, 7, ..., 20** 까지의 연속된 정수 값을 가집니다.
        
    - 종속 변수 **Y**는 선형 관계식 **`y = 10 + 1*x + e`** 를 따릅니다.
        
    - 오차항 **`e`**는 **평균이 0이고 표준편차가 3인 정규분포(`e ~ N(0, 3)`)**에서 추출된 난수입니다.
        
    - 데이터 생성 과정을 명확히 이해할 수 있도록 `numpy` 라이브러리를 사용해 주세요.
        
2. **선형 회귀 모델 적합:**
    
    - 생성된 X, Y 데이터를 사용하여 `statsmodels` 라이브러리로 단순 선형 회귀 모델을 적합합니다.
        
    - 상수항(intercept)이 모델에 포함되도록 해주세요.
        
3. **결과 출력 및 해석:**
    
    - 적합된 모델의 **요약 결과(`summary()`)**를 전체 출력합니다.
        
    - 요약 결과에서 **회귀 계수(기울기)의 p-value**를 확인하여, 기울기가 통계적으로 유의미한지에 대한 해석을 주석으로 추가해 주세요. (예: "p-value가 0.05보다 작으므로, 유의수준 5% 하에서 기울기는 통계적으로 유의미합니다.")
        
4. **새로운 값에 대한 예측:**
    
    - 독립 변수 **X가 30일 때**의 종속 변수 Y 값에 대한 **95% 신뢰구간(Confidence Interval)**과 **95% 예측구간(Prediction Interval)**을 계산하고, 그 결과를 알아보기 쉽게 출력합니다.
        
5. **코드 작성 스타일:**
    
    - **상세한 주석:** Python 초보자도 코드의 각 줄이 어떤 역할을 하는지 쉽게 이해할 수 있도록, 단계별로 상세한 설명을 한글 주석으로 달아주세요.
        
    - **가독성:** 논리적인 흐름에 따라 코드 블록을 구분하고, 변수명은 그 의미를 명확하게 알 수 있도록 작성해 주세요. (예: `independent_variable`, `predicted_value`)
        
    - **필요 라이브러리:** 코드 실행에 필요한 모든 라이브러리(`numpy`, `pandas`, `statsmodels` 등)는 코드 최상단에서 `import` 해주세요.
        

### 초보자를 위한 Python 및 VS Code 실행 가이드

위 프롬프트로 생성된 Python 코드를 처음 실행해 보는 분들을 위해, 아래와 같이 환경 설정부터 실행까지의 과정을 안내합니다.

**1. Python 설치**

- **공식 홈페이지 접속:** [python.org](https://www.python.org/ "null")에 접속하여 최신 버전의 Python 설치 파일을 다운로드합니다.
    
- **설치 시 주의사항:** 설치 프로그램을 실행할 때, 첫 화면에서 **"Add Python to PATH"** 또는 **"Add python.exe to PATH"** 체크박스를 **반드시 선택**하고 설치를 진행하세요. 이 옵션을 선택해야 명령 프롬프트나 터미널 어디에서든 `python` 명령어를 사용할 수 있습니다.
    

**2. Visual Studio Code (VS Code) 설치**

- **공식 홈페이지 접속:** [code.visualstudio.com](https://code.visualstudio.com/ "null")에 접속하여 본인의 운영체제(Windows, Mac 등)에 맞는 설치 파일을 다운로드하여 설치합니다.
    
- **Python 확장 프로그램 설치:**
    
    1. VS Code를 실행합니다.
        
    2. 왼쪽 사이드바에서 네모 블록 모양의 **확장(Extensions)** 아이콘을 클릭합니다.
        
    3. 검색창에 `Python`을 입력하고, Microsoft에서 제공하는 공식 Python 확장 프로그램을 찾아 **Install** 버튼을 누릅니다.
        

**3. 프로젝트 준비**

1. 분석 작업을 저장할 **새 폴더**를 컴퓨터에 만듭니다. (예: `C:\python_study`)
    
2. VS Code를 열고, 상단 메뉴에서 `파일(File)` > `폴더 열기(Open Folder)`를 선택하여 방금 만든 새 폴더를 엽니다.
    
3. 왼쪽 탐색기 패널에서 **새 파일 아이콘**을 클릭하여 `linear_regression.py`와 같은 이름으로 파이썬 파일을 생성합니다.
    
4. 생성된 `linear_regression.py` 파일에 AI가 만들어준 코드를 그대로 복사하여 붙여넣습니다.
    

**4. 필요 라이브러리 설치**

코드를 실행하려면 데이터 분석에 필요한 도구(라이브러리)들을 먼저 설치해야 합니다.

1. VS Code 상단 메뉴에서 `터미널(Terminal)` > `새 터미널(New Terminal)`을 클릭하여 화면 하단에 터미널 창을 엽니다.
    
2. 터미널에 아래 명령어를 한 줄씩 입력하고 엔터(Enter)를 눌러 라이브러리를 설치합니다.
    
    ```
    pip install numpy
    pip install pandas
    pip install statsmodels
    pip install matplotlib
    ```
    
    - `pip`는 파이썬의 패키지 관리자로, 필요한 라이브러리를 쉽게 설치해 주는 명령어입니다.
        

**5. 코드 실행 및 결과 확인**

모든 준비가 끝났습니다. 터미널에서 아래 명령어를 입력하여 코드를 실행합니다.

```
python linear_regression.py
```

- 코드가 정상적으로 실행되면, 터미널 창에 선형 회귀 분석 요약 결과와 X=30일 때의 신뢰구간, 예측구간이 출력될 것입니다.
