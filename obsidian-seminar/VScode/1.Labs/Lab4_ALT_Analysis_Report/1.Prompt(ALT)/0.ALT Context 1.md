## 가속수명시험(ALT) 데이터 분석 절차

가속수명시험(ALT) 데이터 분석의 최종 목표는 가혹한 조건에서 단시간에 얻은 수명 데이터를 사용하여, 실제 사용 조건에서의 제품 수명 분포를 추정하고 **B수명(B-Life)**이나 **고장 확률**과 같은 신뢰성 척도를 계산하는 것입니다. 분석은 다음과 같은 체계적인 절차에 따라 진행됩니다.

### **1단계: 데이터 준비 및 탐색 (Data Preparation & Exploration)**

분석을 시작하기 위해 가장 먼저 데이터를 불러오고 구조를 파악해야 합니다.

- **데이터 로딩**: `csv` 파일과 같이 정형화된 데이터를 불러옵니다. 데이터는 일반적으로 다음과 같은 컬럼으로 구성됩니다.
    
    - `stress`: 온도, 전압, 습도 등 가속 스트레스 수준
        
    - `time`: 고장이 발생했거나 시험이 중단된 시간
        
    - `cens`: 데이터의 성격을 나타내는 관측 중단(Censoring) 정보.
        
        - **고장 데이터 (Failure Data)**: 정해진 시간 안에 고장이 발생한 데이터 (예: `cens` = 1)
            
        - **우측 관측 중단 데이터 (Right-Censored Data)**: 정해진 시험 시간이 다 될 때까지 고장 나지 않은 데이터 (예: `cens` = 0)
            
- **데이터 전처리**: 불러온 데이터를 분석에 용이한 형태로 가공합니다.
    
    - 고장 데이터와 관측 중단 데이터를 별도의 데이터프레임으로 분리합니다.
        
    - 각 데이터를 스트레스 수준(`stress`)에 따라 그룹으로 나눕니다. 이는 각 스트레스 조건별로 수명 분포를 분석하기 위함입니다.
        

### **2단계: 수명분포 선정 (Life Distribution Selection)**

각 스트레스 수준에서 얻은 수명 데이터가 어떤 확률 분포를 따르는지 결정합니다. 이는 후속 분석의 정확성을 위해 매우 중요한 단계입니다.

- **후보 분포**: 와이블(Weibull) 분포, 대수정규(Lognormal) 분포 등이 대표적인 수명분포로 사용됩니다.
    
- **분석 방법**:
    
    1. **시각적 방법 (확률지, Probability Plot)**: 각 후보 분포의 확률지에 데이터를 타점하여 데이터가 직선에 가깝게 배열되는지 확인합니다.
        
    2. **통계적 방법 (Goodness-of-Fit)**: 최대우도추정(MLE)과 같은 통계적 방법으로 각 분포의 모수(parameter)를 추정합니다. 이후, 로그-우도(Log-likelihood), AIC(Akaike Information Criterion), AD(Anderson-Darling) 값 등을 비교하여 가장 데이터에 적합한 분포를 선정합니다.
        

### **3단계: 가속성 검증 (Accelerability Test)**

가속 조건에서의 고장 메커니즘이 사용 조건에서도 동일하게 유지되는지를 확인하는 과정입니다. 가속성이 성립해야만 가속 시험 결과를 사용 조건으로 외삽(extrapolate)할 수 있습니다.

- **가정**: 수명 분포의 **형상모수(shape parameter)**가 모든 스트레스 수준에서 동일해야 합니다.
    
    - **와이블 분포**: 형상모수 베타(β)가 모든 스트레스 수준에서 동일해야 합니다.
        
    - **대수정규 분포**: 형상모수 시그마(σ)가 모든 스트레스 수준에서 동일해야 합니다.
        
- **검증 방법**:
    
    1. **시각적 방법 (확률지)**: 2단계에서 그린 확률지에서 각 스트레스 수준별로 그려진 **직선들이 서로 평행한지** 확인합니다. 확률지의 기울기는 형상모수를 나타내기 때문입니다.
        
    2. **통계적 방법 (우도비 검정, Likelihood Ratio Test)**: 통계적으로 형상모수가 동일한지에 대한 가설 검정을 수행합니다.
        
        - **귀무가설 (H0​)**: 모든 스트레스 수준에서 형상모수는 동일하다 (β1​=β2​=⋯=βk​).
            
        - **판정**: 검정 결과 p-value가 유의수준(예: 0.05)보다 크면 귀무가설을 채택하여, **가속성이 성립**한다고 판단합니다.
            

### **4단계: 가속수명모델 적합 (Accelerated Life Model Fitting)**

가속성이 확인되면, 스트레스와 수명 간의 관계를 설명하는 물리적/경험적 모델을 수립합니다.

- **수명-스트레스 관계식 선택**: 스트레스의 종류에 따라 적합한 모델을 선택합니다.
    
    - **아레니우스 (Arrhenius)**: 온도 스트레스에 주로 사용됩니다.
        
    - **역거듭제곱 (Inverse Power)**: 전압, 압력 등 비열 스트레스에 사용됩니다.
        
    - **온습도 모델**: 온도와 습도가 복합적으로 작용할 때 사용됩니다.
        
- **모델링**: 2단계에서 선정한 수명분포와 4단계에서 선택한 수명-스트레스 모델을 결합하여 전체 데이터를 가장 잘 설명하는 **가속수명모델의 파라미터를 추정**합니다.
    
    - 예: 와이블-아레니우스 모델은 와이블 분포의 척도모수(η)가 온도(T)에 대한 아레니우스 관계식을 따른다고 가정합니다.
        
        - η(T)=exp(α+kEa​​⋅T1​)
            

### **5단계: 사용조건 수명 예측 및 신뢰성 척도 계산**

완성된 가속수명모델을 사용하여 최종 목표인 **정상 사용 조건(use condition)에서의 수명을 예측**합니다.

- **사용조건 수명분포 추정**: 수립된 가속수명모델에 정상 사용 조건의 스트레스 값을 대입하여, 해당 조건에서의 수명분포 모수(예: 와이블 분포의 척도모수 η와 공통 형상모수 β)를 계산합니다.
    
- **주요 신뢰성 척도 산출**: 추정된 수명분포를 바탕으로 다양한 신뢰성 척도를 계산합니다.
    
    - **B수명 (B-Life / Percentile)**: 모집단의 p%가 고장 나는 시간. 예를 들어, **B10 수명**은 10%의 제품이 고장 날 때까지의 시간입니다.
        
    - **누적고장확률 (CDF)**: 특정 시간 `t`까지 제품이 고장 날 확률을 계산합니다.
        
    - **가속계수 (Acceleration Factor, AF)**: 정상 사용조건에서의 수명이 가속 조건보다 몇 배 더 긴지를 나타내는 비율입니다.
