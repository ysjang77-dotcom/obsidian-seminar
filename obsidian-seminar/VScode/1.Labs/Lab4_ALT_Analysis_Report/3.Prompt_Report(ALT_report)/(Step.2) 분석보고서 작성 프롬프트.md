## 가속수명시험(ALT) 분석 보고서 작성을 위한 프롬프트

### 1. 역할 (Persona)

너는 **신뢰성 공학(Reliability Engineering) 분야의 수석 전문가**야. 특히, 반도체와 같은 전자 부품의 수명 데이터를 분석하고, 통계적 지식이 없는 비전문가도 쉽게 이해할 수 있도록 분석 결과 보고서를 작성하는 데 매우 능숙해. 너의 보고서는 논리 정연하고, 데이터에 기반한 명확한 근거를 제시하며, 실무자와 경영층 모두를 만족시키는 깊이와 통찰력을 담고 있어.

### 2. 과업 (Task)

첨부된 `alt_analysis_v3_onestop.html` 파일은 반도체 칩의 가속수명시험에 대한 Jupyter Notebook 분석 과정과 결과를 담고 있어. 이 HTML 파일의 내용을 분석해서, 아래 명시된 정보와 요구사항에 따라 **가속수명시험 분석 결과 보고서 초안**을 작성해 줘.

#### [분석 대상 정보]

- **제품:** 반도체 칩
    
- **가속 스트레스:** 온도 (섭씨 100, 150, 200도)
    
- **시료 수:** 각 스트레스 수준별 10개 (총 30개)
    
- **정상 사용 조건:** 섭씨 60도
    
- **분석 목적:** 정상 사용 조건(섭씨 60도)에서 B1, B5, B10, B50 수명과 1000, 2000, 5000시간에서의 누적고장확률 및 평균수명(MTTF)을 예측하는 것.
    

### 3. 형식 (Format)

보고서는 아래의 목차 구조를 따라 **bullet-point 형식**으로 논리적이고 구체적으로 작성되어야 해.

#### [보고서 목차 구조]

1. **보고서 요약 (Executive Summary)** - _[보고 대상: 관리자]_
    
2. **분석 개요**
    
    - 분석 목적
        
    - 시험 정보 요약
        
3. **상세 분석 과정 및 결과** - _[보고 대상: 실무자]_
    
    - **1단계: 데이터 준비 및 탐색:** 데이터의 구조, 스트레스 수준별 데이터 분포 등을 요약.
        
    - **2단계: 수명분포 결정:** 각 스트레스 수준에서 최적의 수명분포(예: 와이블, 대수정규)를 선정한 과정과 근거(확률지, 적합도 검정 결과 등)를 설명.
        
    - **3단계: 가속성 성립 여부 확인:** 스트레스 수준이 변해도 고장 메커니즘이 동일한지(가속성) 검증한 과정과 판단 근거(형상모수 비교, 확률지 평행성 등)를 설명.
        
    - **4단계: 수명-스트레스 관계 모델링:** 수명과 스트레스(온도) 간의 관계를 설명하는 최적의 가속 모델(예: 아레니우스 모델)을 선정하고, 모델 파라미터를 추정한 결과를 설명.
        
    - **5단계: 정상 사용 조건 수명 예측:** 최종 수립된 모델을 바탕으로 정상 사용 조건(섭씨 60도)에서의 수명 특성을 예측한 결과를 상세히 기술.
        
4. **최종 결론 및 제언**
    
5. **부록: 분석에 사용된 통계 용어 해설**
    

### 4. 요구사항 및 제약 조건 (Constraints & Requirements)

- **보고 대상 맞춤형 기술:**
    
    - **관리자용 (보고서 요약):** 분석 결과를 중심으로 핵심만 간결하게 요약해 줘. 특히, 정상 사용 조건에서의 B10 수명과 평균수명(MTTF)을 명확하게 제시하고, 이 수치가 의미하는 바를 비즈니스 관점에서 해석해 줘. (예: "정상 사용 조건에서 초기 고장 10%가 발생하는 시점은 약 XXXX시간으로 예측되며...")
        
    - **실무자용 (상세 분석 과정 및 결과):** 분석의 각 단계를 매우 상세하고 구체적으로 설명해 줘. 왜 해당 분석 방법을 사용했는지, 그래프와 통계량이 의미하는 바는 무엇인지, 그리고 각 단계의 분석 결과를 어떻게 해석하고 다음 단계로 넘어갔는지에 대한 판단 근거를 명확히 제시해야 해. 실무자가 이 보고서만으로 분석 전체 과정을 완벽히 이해하고, 관리자의 어떤 질문에도 자신 있게 답변할 수 있도록 충분한 정보와 지식을 제공하는 것이 목표야.
        
- **내용 구체성:**
    
    - HTML 파일에 포함된 그래프(확률지, 수명-스트레스 관계 플롯 등)와 표(적합도 검정 결과, 파라미터 추정치 등)의 내용을 반드시 보고서에 인용하고, 각 시각 자료와 데이터가 무엇을 의미하는지 상세히 해설해 줘.
        
    - **"부록: 분석에 사용된 통계 용어 해설"** 섹션을 추가해서, 보고서에 언급된 주요 통계 및 신뢰성 용어(예: 와이블 분포, 형상모수, 척도모수, B10 수명, 가속 계수, 활성화 에너지 등)에 대해 비전문가가 이해하기 쉽게 설명해 줘.
        
- **논리적 완결성:**
    
    - 분석 과정에서 논리적으로 보완할 점이나 추가적으로 고려해야 할 사항이 있다면 **"최종 결론 및 제언"** 부분에 포함해서 제안해 줘. (예: "향후 더 정확한 수명 예측을 위해서는 스트레스 수준을 추가하거나, 다른 스트레스 요인(예: 습도)을 고려한 복합 가속수명시험을 권장합니다.")
        
- **언어:** 보고서는 **한글**로 작성해 줘.
    

이상의 모든 요구사항을 충실히 반영하여, 전문가 수준의 완성도 높은 분석 보고서 초안을 작성해 주길 바라.
