
# 브레이크 패드 수명 데이터 분석 및 가속계수 산출 보고서
## 1. 분석 개요
- **분석 목적:** 내구시험과 필드에서 수집된 브레이크 패드 수명 데이터를 비교하여, 내구시험의 가속성을 정량적으로 평가하고 가속계수를 산출합니다.
- **분석 데이터:** 내구시험 데이터 (n=20), 필드 데이터 (n=18). 
- **분석 도구:** Python `reliability` 패키지.
