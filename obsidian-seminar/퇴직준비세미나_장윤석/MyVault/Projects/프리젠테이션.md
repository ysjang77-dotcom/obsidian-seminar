## Step 1: RCIF 프롬프트 작성

  당신은 한국기계연구원의 수석연구원이자 20년 경력의 기술 프레젠테이션 전문가입니다.

  이번에 개발한 'AI 기반 예지보전 시스템'의 성과를 경영진에게 보고하기 위한
  발표자료 구조를 만들어주세요.

  제약 조건:
  - 슬라이드 분량: 8장 이내
  - 핵심 타겟: 기술적 세부사항보다는 경제적 효과와 도입 타당성에 관심이 많은 경영진
  - 포함 내용: 기존 방식의 문제점, AI 도입 후 효율성 향상 지표, 향후 로드맵

  각 슬라이드별로 제목, 핵심 메시지 3가지, 추천 시각자료, 스크립트 초안을
  마크다운 표 형식으로 정리해주세요.

## Step 3: 이 결과가 좋은 이유 분석

  주목할 점이 3가지 있습니다:

  1. 청중 맞춤 구성
  - 기술 원리 설명은 슬라이드 1장(3번)으로 최소화
  - 경영진이 관심 있는 비용, ROI, 의사결정에 5장을 할애

  2. 스토리 흐름
  문제 제기 → 해결책 제시 → 성과 증명 → ROI → 확산 계획 → 승인 요청
  논리적으로 "왜 이걸 계속해야 하는가?"를 설득하는 구조입니다.

  3. 구체적 숫자
  - "효과가 있었습니다" (X) → "비계획 정지 68% 감소, 연 4.8억 절감" (O)
  - AI가 생성한 숫자는 예시이므로, 실제 데이터로 교체하면 됩니다.

## Step 4: Gamma.app으로 슬라이드 자동 생성

### Gamma용 텍스트 변환 결과

Step 2의 구조를 Gamma.app 'Text to Deck'에 넣을 수 있는 형태로 변환:

```
# AI 예지보전 시스템 성과 보고
프로젝트 개요 | 목표: 설비 가동률 향상 | 기간: 2025.3~2026.1

# 현재의 문제: 사후 정비의 한계
- 연간 비계획 정지 47건 발생
- 정비 비용 연 12억 원
- 생산 손실 연 8억 원

# 해결책: AI 예지보전이란?
- 센서 데이터 실시간 분석
- 고장 72시간 전 예측
- 계획 정비로 전환

# 도입 성과: 숫자로 증명
- 비계획 정지 68% 감소
- 정비 비용 40% 절감
- 설비 가동률 92% → 97%

# 투자 대비 효과 (ROI)
- 개발비 3억 원 투입
- 연간 절감 효과 9.6억 원
- 투자회수 기간 4개월

# 타 사업장 확산 계획
- 2단계: 창원 공장 (2026 하반기)
- 3단계: 전 사업장 (2027)
- 예상 절감: 연 50억 원

# 경영진 의사결정 요청
- 2단계 예산 5억 원 승인 요청
- 전담 인력 2명 배정
- 2026년 6월 착수 목표

# Q&A / 감사합니다
핵심 KPI 요약 | 담당자 연락처 | 별첨 자료 안내
```

### 실습 순서
1. gamma.app 접속 → 구글 계정 로그인
2. 'Create new with AI' → 'Text to Deck' 선택
3. 위 텍스트 붙여넣기
4. 언어: Korean, 스타일: Professional
5. 테마 선택 → Generate 클릭
6. Step 2의 "추천 시각자료" 참고하여 차트/이미지 추가

### 핵심 워크플로우
```
[RCIF 프롬프트] → [Claude 구조 생성] → [Gamma.app 붙여넣기] → [자동 생성] → [수정/보완]
```
전체 소요시간: 약 15~20분 (기존 9시간 대비)
