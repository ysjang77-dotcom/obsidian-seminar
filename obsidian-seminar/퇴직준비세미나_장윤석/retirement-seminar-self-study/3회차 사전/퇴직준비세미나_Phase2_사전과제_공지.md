## Phase 2 세미나 소개

Phase 2의 주제는 **'문서작성이 사라진다 - 지식 큐레이션'**입니다.

> "기록은 내가 하고, 정리는 AI가 한다.
> AI Agent가 나의 반복 업무를 대신하면, 비로소 창의적 업무에 집중할 수 있습니다."

Phase 2에서는 AI Agent의 핵심 개념을 이해하고, Claude Code를 활용하여 실제 업무를 자동화하는 경험을 합니다.

| 회차 | 주제 | 핵심 내용 |
|------|------|----------|
| 3회차 | Obsidian - 제2의 뇌 구축 | PARA 방법론, 양방향 링크, 그래프 뷰 |
| 4회차 | Claude Skill을 이용한 자동 문서화 | RAG 기초, MCP 설정, 템플릿 기반 자동 문서화 |

## 왜 Phase 2 과제가 어려운가?

### 1. AI Agent 개념 이해가 필요합니다

단순 대화를 넘어, AI Agent의 동작 원리를 이해해야 합니다.

- **Tool Use**: AI가 파일을 읽고, 쓰고, 검색하는 방법
- **Planning**: AI가 복잡한 작업을 단계별로 분해하는 방법
- **Memory**: AI가 맥락을 기억하고 활용하는 방법

### 2. Claude Code 설치 및 설정이 필요합니다

터미널에서 AI와 협업하는 환경을 구축해야 합니다.

- Claude Code 설치
- 기본 명령어 숙지 및 실행 테스트

### 3. 본인 업무를 분석해야 합니다

남의 예제를 따라하는 것이 아니라, **본인의 반복 업무**를 찾아 분석해야 합니다.

- 어떤 업무가 자동화에 적합한가?
- 입력과 출력은 무엇인가?
- AI가 할 일과 사람이 할 일은 어떻게 나누는가?

### 4. 실제로 실행해봐야 합니다

설계한 내용을 Claude Code로 **직접 실행**하고, 결과를 얻어야 합니다.

- Agent가 예상대로 동작하지 않을 때 프롬프트 수정
- 오류 발생 시 원인 분석 및 재시도
- Agent Loop(요청→계획→실행→결과)의 각 단계 관찰

### Claude Code 자동화 태스크

Claude Code를 사용하여 본인 업무의 반복 작업 1건을 자동화합니다.

| 항목 | 설명 |
|------|------|
| Claude Code 설치 | Claude Code 설치 완료 |
| 자동화 대상 업무 선정 | 본인이 매일/매주 반복하는 작업 1건 |
| Agent화 설계 문서 작성 | 입력-처리-출력 정의, 프롬프트 설계 |
| Agent 실행 및 결과물 획득 | 실제로 Claude Code에서 실행 |
| Agent Loop 로그 캡처 | 요청→계획→실행→결과 4단계 |

### 자동화 대상 업무 예시

| 업무 유형 | 구체적 예시 |
|----------|------------|
| 📊 데이터 정리 | Excel 데이터 정리, CSV 변환, 중복 제거 |
| 📝 문서 생성 | 일일 리포트, 회의록 초안, 보고서 템플릿 |
| 📧 이메일 작성 | 정기 공지, 협조 요청, 결과 공유 메일 |
| 📁 파일 관리 | 파일명 일괄 변경, 폴더 정리, 백업 |
| 🔍 정보 수집 | 웹에서 정보 검색 및 요약, 논문 초록 정리 |

### 제출물

1. **설치 완료 스크린샷** (버전 정보 포함)
2. **Agent화 설계 문서** (업무 분석 + 입력/출력 정의 + 프롬프트)
3. **Agent 실행 로그 스크린샷** (4단계 모두)
4. **최종 결과물** (Agent가 생성한 파일 또는 출력물)

## 과제 수행 시 참고사항

> 과제는 스스로 해결하시는 것을 권장드립니다.

AI 에이전트 활용의 핵심은 **"모르는 것을 스스로 해결하는 능력"**입니다.

- 오류가 발생하면 오류 메시지를 Claude에게 물어보세요
- 예상과 다르게 동작하면 프롬프트를 수정해보세요
- 막히면 YouTube나 공식 문서를 검색해보세요

**이 과정 자체가 AI 시대 생존 스킬의 핵심입니다.**

과제 수행 중 해결하기 어려운 문제가 있으시다면, **세미나 질의응답 시간에 함께 해결**해 드리겠습니다. 먼저 본인 힘으로 시도해보시고, 어떤 시도를 했는지 기록해두시면 더욱 효과적인 학습이 됩니다.

## 과제의 의미

Phase 2 과제를 통해 여러분은:

- [ ] AI를 단순한 챗봇이 아닌 **자동화 도구**로 활용하는 경험
- [ ] Claude Code를 통해 **터미널에서 AI와 협업**하는 방법
- [ ] 본인 업무를 분석하여 **자동화 가능한 부분**을 찾는 안목
- [ ] Agent Loop를 직접 관찰하며 **AI의 사고 과정**을 이해
- [ ] 오류와 시행착오를 통해 **문제 해결 능력** 강화

을 얻게 됩니다.

**이것이 바로 "문서작성이 사라진다"의 실체입니다.**
