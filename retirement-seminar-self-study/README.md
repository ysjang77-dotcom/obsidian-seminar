# 퇴직준비세미나 자율학습자료

AI 시대 생존을 위한 10회차 세미나 자율학습 자료입니다.

## 📚 소개

이 자료는 "**퇴직 = 직무의 소멸에 대비하는 것**"이라는 철학을 바탕으로, AI 시대에 연구자와 전문가가 갖추어야 할 핵심 역량을 단계별로 학습할 수 있도록 구성되었습니다.

### 핵심 컨셉
- **AI를 도구가 아닌 파트너로**: Claude Code와 함께하는 실무 중심 학습
- **제2의 뇌 구축**: Obsidian + PARA 시스템으로 지식 관리
- **T자형 역량 확장**: 깊은 전문성 + 넓은 AI 활용 능력

## 🚀 사용 방법

### 1. 필수 환경 설정
- **Obsidian**: [obsidian.md](https://obsidian.md) 에서 다운로드
- **Claude Code**: Anthropic API 키 필요 ([console.anthropic.com](https://console.anthropic.com))
- **Git**: 버전 관리용 (선택사항)

### 2. 자료 활용
1. 이 저장소를 클론하거나 다운로드합니다:
   ```bash
   git clone https://github.com/orientpine/retirement-seminar-self-study.git
   ```
2. Obsidian에서 `회차별자료` 폴더를 Vault로 열거나 기존 Vault에 복사합니다.
3. 1회차부터 순서대로 학습합니다.

### 3. 학습 순서
각 회차는 이전 회차의 내용을 기반으로 합니다. **순서대로 학습**하는 것을 권장합니다.

| 단계 | 회차 | 핵심 역량 |
|------|------|-----------|
| 기초 | 1-2회차 | 환경 설정, 프롬프트 기초 |
| Agent | 3-4회차 | Agent 개념, 자동화 워크플로우 |
| 지식관리 | 5-6회차 | Obsidian, Skill 시스템 |
| 도구제작 | 7-8회차 | MCP 도구, 데이터 분석 |
| 관리/종합 | 9-10회차 | 프로젝트 관리, 역량 확장 |

### 4. 실습 방법
각 문서에는 다음이 포함되어 있습니다:
- 🎯 **학습 목표**: 체크리스트 형태의 목표
- 📋 **강의 개요**: 3시간 분량 구성
- 💡 **실습 과제**: 직접 해볼 수 있는 과제
- ✅ **이해도 체크**: 자가 점검 질문
